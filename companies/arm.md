# Arm (剑桥) 实习情报

> Arm Holdings (软银控股,2023 NASDAQ IPO) 是全球 IP 半导体核心,几乎所有 ARM 架构 SoC 的根源都在剑桥 Fulbourn Road 总部。Arm 是英国"硬科技"实习的旗舰品牌,**placement 文化极成熟**,有完整 3 / 6 / 12 个月 + 两年 Part-time Undergraduate Programme,Chinese / Indian 学生历史招聘量大。

## 1. 公司速览

| 项目 | 内容 |
|---|---|
| 总部 | 剑桥 Cambridge UK (Peterhouse Technology Park / Fulbourn Road) |
| 其他 UK 据点 | Manchester、Sheffield、Belfast、Warwick |
| 主营业务 | CPU IP (Cortex)、GPU IP (Mali)、NPU、AI ML、Server Chips (Neoverse) |
| 技术栈 | C / C++ / SystemVerilog / Python / Assembly,Linux kernel,LLVM/GCC |
| 实习时长 | **3 / 6 / 12 个月 placement** + 两年 Part-time Undergraduate Programme |
| 申请窗口 | 秋季 **9-10 月开放**,Rolling,12 月-1 月趋势上招满 |
| 起薪 | 实习: £22-26k (基础) 到 £34k+ (Master / 12 month) |

## New Grad / Junior 岗位

> 信息丰富度: ⭐⭐⭐⭐⭐ (Arm Graduate Programme 是英国 placement / graduate 文化的旗舰之一, 中英印社区面经齐全, Glassdoor / Bright Network / 一亩三分地数据厚)
> 区别于实习: 全职 entry-level (0-2 年经验); Arm 有**最成熟的命名 Graduate Programme** — Arm Graduate Engineering Programme, 2 年结构化, 跨多个 engineering function

- **常见岗位名**: Graduate Software Engineer、Graduate Hardware / Silicon Engineer、Graduate ML Engineer、Graduate Architecture Engineer、Graduate Research Engineer; 招聘页明确按 "Graduate" 命名独立 req
- **欧洲地点**: 总部 **Cambridge UK (Peterhouse Tech Park / Fulbourn Road)** — 最大 Graduate cohort 入口; 其他 UK: **Manchester、Sheffield、Belfast、Warwick**; 欧洲外: Sophia Antipolis (法国, 小)、Lund (Sweden, 小); Graduate cohort 90% 在 UK
- **是否有独立 Graduate Programme**: **Yes — Arm Graduate Engineering Programme**
  - **2 年结构化**, 包含 rotation across teams (option to switch function 一次)、mentorship、Arm Academy 培训
  - 多个 track: Software (Compilers / OS / Tools)、Silicon (CPU / GPU / SoC)、Machine Learning、Architecture、Research
  - cohort 集中 onboarding (通常 9 月入职), 大规模 grad batch 80-150 人/年
  - 满 2 年默认 convert 为 Engineer (band 内升级), 大量 Intern → Graduate → Engineer 通道
- **非EU签证 (全职)**:
  - **UK Skilled Worker Visa**: Arm UK 是持牌 sponsor; Graduate role **会 sponsor Skilled Worker** (2024-04 起 going rate ≥ £38,700, new entrant 阈值约 £30,960); Graduate 起薪通常达标
  - **Graduate Visa (PSW)**: UK 毕业生 2 年 (PhD 3 年), Arm 大量接受 PSW candidate 后再转 Skilled Worker
  - **High Potential Individual (HPI) Visa**: Top 50 全球大学 5 年内毕业可申请 2 年 (PhD 3 年), Arm 接受
  - 实习多用 Student Visa, 全职 sponsor 是常规操作 — Arm 是 UK **国际学生 sponsor 最友好的硬科技公司之一**, 历史招大量中国 / 印度 candidate
- **薪资范围 (base, gross/年)** (Glassdoor / Levels.fyi / Bright Network 2024-2025):
  - **Cambridge Graduate SWE**: base ~£36,000-44,000, +sign-on £2-3k + 10% bonus + RSU (Arm 2023 NASDAQ IPO 后 RSU 价值实化, 2024-2025 股价从 $50 涨到 $150+), TC ~£42k-55k
  - **Cambridge Graduate Hardware / Silicon**: base ~£36,000-44,000 (与 SWE 持平)
  - **Cambridge Engineer (1-2 yrs)**: base ~£48,000-62,000, TC ~£60k-85k (含 RSU)
  - **Manchester / Sheffield / Belfast**: base ~£32,000-40,000 (生活成本低)
  - 福利: RSU、年度 bonus、pension 18% (业内极高)、健康保险、健身、Arm Academy 培训、relocation budget (国际 candidate ~£3-5k)
  - 比 FAANG London (Google L3 ~£70k base + RSU) 低, 但与 Cambridge 本地 (ARM / Microsoft Research Cambridge / DeepMind Cambridge) 比有竞争力, WLB 极佳
- **申请窗口**: **秋季 9-10 月开放** Rolling 评估 (先到先评), 通常 **12 月 - 次年 1 月趋势上招满**; 入职 9 月 (next September); cohort batch 集中 onboarding
- **面试流程差异 vs 实习**:
  - 实习 4 轮 (OA → tech chat → tech panel → manager), Graduate **5-6 轮 + Final Day onsite assessment**
  - **Final Day onsite** 含: 2 轮 tech panel + behavioural + 团队 fit (整天)
  - System Design / 架构题在 Graduate 更深: SWE 会考 compiler pass design / OS scheduler / cache coherence; Silicon 会考 pipeline hazard + memory consistency + power 优化
  - HireVue 录像 + 性格测试在 Graduate 流程仍保留 (实习一致)
  - 与 FAANG 不同: Arm 极少考 LeetCode hard, 偏重 systems / architecture / C++ 深度
- **录取竞争**: Arm Graduate 录取率 ~5-10% (申请池 5000+/年, cohort 100-150 人), bar 中等偏低 (vs FAANG), 重深度 systems / hardware 知识; 国际生 friendliness **业内顶尖** — UK 硬科技 grad 国际生最佳入口, 大量中国 / 印度 / 东欧 candidate; 转正率高 (实习 → Graduate → Engineer 通道顺畅)

## 2. 实习岗位与方向

Arm 招聘按 **Engineering Function** 划分,每个 function 都有 Internship + Placement 双入口:

| 岗位类别 | 团队 / 方向 | 技术栈 |
|---|---|---|
| **Software Engineering Intern** | Compilers (LLVM/GCC)、OS Kernel、Tools (Mali Graphics)、Productivity Eng | C++、Python、Linux |
| **Hardware / Silicon Intern** | CPU (Cortex)、GPU (Mali)、SoC Integration、Physical Design | SystemVerilog、VHDL、TCL,Synopsys/Cadence EDA |
| **Machine Learning Intern** | ML Engineering、NPU IP、Compiler Optimization for ML | Python、PyTorch、TVM、MLIR |
| **Architecture Intern** | Architecture Technology Group、Performance Modeling | C++、Performance simulators |
| **Research & Education Placement** | Arm Research (Cambridge / Manchester) | 研究方向多样 |

## 3. 申请要求与签证

- **学历**: STEM 学位在读,本科 / 硕士均可;计算机 / EE / 物理背景
- **签证关键**:
  - Arm UK 是 Skilled Worker 持牌
  - **多数 placement 接受国际学生 (Student Visa 工作权)** — 历史上对 Chinese / Indian 学生开放程度极高
  - 12 个月 placement 必须能在 UK 学习 / 居留期间合法工作 (Tier 4 / Student Visa)
  - 非 UK 学生若需自带 sponsor,部分岗位可走 GAE / T5 但难度高
- **薪资 (Glassdoor / Indeed)**:
  - SWE Intern: 平均 ~£25,566 / 年 pro-rated
  - 25-75 percentile: £19.3k - £34.9k
  - 部分 Master / 12 month placement 可上 £30k+

## 4. 面试流程

```
1. Online Application + CV + Cover Letter
2. Online Assessment (HireVue 录像 + 性格 / 逻辑测试)
3. 技术 / 招聘官 chat (30-45 min)
4. Technical interview (1-2 轮 panel)
   - SWE: C / C++ 题、内存管理、Linker、Linux 系统编程
   - Silicon: Verilog、流水线 hazard、CDC、setup/hold
   - ML: 框架细节 (PyTorch internals)、模型量化、kernel fusion
5. Manager + Behavioral 终轮
```

通常 **3-5 周** 完成,Rolling 评估,先到先评。

## 6. 文化与口碑

- **正向**:
  - **极佳的 mentor 制**: 95% 实习生说技能提升明显
  - 工程深度强 — 接触真实的 silicon production 流程
  - WLB 一流: 9-5 节奏、上下班时间灵活、剑桥城市适合学习
  - 转正率高 — 大量 Intern → Graduate Programme 直通
  - 国际学生友好度业内顶尖
- **挑战**:
  - 升职慢 — Big Tech 节奏对比下职业上升慢
  - 薪资天花板低于 FAANG
  - 部分团队遗留 C++03 / Perl 老代码

## 7. 重点准备清单

### Software / Compiler / Tools 方向
1. **C / C++ 深入**: 指针、内存对齐、Smart Pointer、RAII、Rule of Five
2. **OS / Linux 系统编程**: fork/exec、signal、mmap、syscall
3. **DSA**: LC Easy-Medium,树 / 图遍历高频
4. **Compiler 基础 (相关岗位)**: LLVM IR、SSA、寄存器分配
5. **架构知识**: ARM ISA 基础、流水线、cache、内存层次

### Silicon / Hardware 方向
1. **Verilog / SystemVerilog**: 时钟域、阻塞 / 非阻塞、CDC、metastability
2. **数字逻辑**: setup/hold、状态机、PLL
3. **EDA 工具**: Synopsys (DC、PrimeTime)、Cadence Innovus 基础概念

### ML 方向
1. **PyTorch internals**: autograd、CUDA kernel、custom op
2. **量化 / 剪枝**: INT8 quantization、QAT vs PTQ
3. **Compiler for ML**: TVM、MLIR、Triton
