# Delivery Hero 实习情报（Berlin HQ）

> 范围：Software Engineering Intern、Data Analyst / Data Engineering Intern。Delivery Hero 是全球 70+ 国家运营的食品 / 杂货配送巨头，旗下品牌包括 foodpanda（亚洲）、Talabat（中东）、PedidosYa（南美）、Glovo（南欧 / 拉美，2022 完全合并）、Yemeksepeti（土耳其）等，工程总部主要在 **Berlin Mitte**。

## New Grad / Junior 岗位

> ⚠️ **待 web 验证**: 本节未做实时 WebSearch 与 WebFetch (agent sandbox 限制), 数据完全基于模型已有知识 + 文件 intern 部分提取; 引用链接为推测入口, 未直接核对. **二次核实前勿用于薪资 / 截止日期决策**.

> 信息丰富度: ⭐⭐⭐⭐ (Tech Graduate Program 官方信息完整, Berlin Junior SWE Glassdoor / Levels.fyi 数据中等)
> 区别于实习: 全职 entry-level (0-2 年经验); 有结构化的 **Tech Graduate Program (TGP)** 入口 + 直接 Junior Engineer 入口两条线

- **常见岗位名**: Junior Software Engineer、Junior Data Engineer / Analyst、Junior ML Engineer; **Tech Graduate Program (TGP)** 是命名最响亮的入口 — 6 个月结构化 graduate program (含 upskilling sessions / mentor / capstone), 满 6 个月后默认 convert 为 Junior Engineer
- **欧洲地点**: 全职工程主要在 **Berlin Mitte HQ**; 其他欧洲品牌 (Talabat Dubai、foodpanda 亚太、PedidosYa 拉美) 是本地 entity 招聘, 走的不是 Berlin 流程; Junior SWE 在欧洲基本默认 Berlin
- **是否有独立 Graduate Programme**: **Yes — Tech Graduate Program (TGP)**, 这是 11 家中少数几家有命名 grad scheme 的
  - 6 个月全职, 春秋两批 (常见 March / September intake)
  - 含 upskilling sessions、社交活动、mentorship、12 周 capstone 项目
  - 资格: CS / DE 或相关 Bachelor / Master, 最多 **2 年工作经验**, 2 天/周 Berlin Mitte onsite
  - 满 6 个月默认 convert 为 Junior Engineer (除非 performance fail)
- **非EU签证 (全职)**:
  - **EU Blue Card (Germany)**: TGP 全职 → Delivery Hero **会 sponsor Blue Card**, 提供 relocation budget; 2025 阈值 ~€45,300/年, TGP 起薪通常达标
  - 与实习政策对比明显: 实习不 sponsor, 但 TGP 是海外 candidate 绕过实习签证卡点的最佳入口
  - 招聘官在 recruiter call 会主动确认 visa / relocation 需求
- **薪资范围 (base, gross/年)** (Glassdoor / Levels.fyi 2024-2025):
  - **Berlin Tech Graduate Program**: base ~€55,000-65,000 (对标 Berlin Junior SWE)
  - **Berlin Junior SWE (直接入口)**: base ~€55,000-68,000, TC ~€58k-72k (含 bonus, 通常无 RSU)
  - **Berlin Mid-level SWE (1-2 yrs)**: base ~€68,000-85,000
  - 福利: Pluxee benefits、餐券、内购食堂、健身房折扣、心理咨询、relocation budget (TGP / Junior 全职)
  - 与 Zalando 相比稍低, 与 N26 接近, 远低于 Berlin 顶部 fintech (Trade Republic / Mollie)
- **申请窗口**: **TGP 春秋两批** (常见 March / September cohort), 应用窗口提前 4-6 个月 (前一年秋 / 前一年春); 直接 Junior SWE req 是 rolling 全年
- **面试流程差异 vs 实习**:
  - 实习 / TGP 流程相似: 4 轮 (Recruiter → Engineer R1 live coding → Engineer R2 概念 → Senior Engineer behavioral)
  - 全职 Junior SWE 比 TGP **多 1 轮 system design** (rate limiter / Kafka consumer / 缓存策略 / pagination)
  - HM round 会问 1-2 年 career 规划
  - 比 Zalando / Klarna 流程短 (2-4 周) — 是 Berlin 大厂中流程最友好的
- **录取竞争**: DH 在巨头中是公认**门槛较低 + 流程友好 + 高 volume**; TGP 春秋两批每批约 20-40 人; 国际生 friendliness 较高 — TGP 是少数明确 sponsor 全职 grad 的 Berlin 项目, 印度 / 中国 / 土耳其 / 巴西 candidate 历史录取多; AdTech / Global Discovery / Q-Commerce / Fintech 是 2024-2025 hc 重点

## 2. 岗位画像

### 2.1 Software Engineering Intern
- 技术栈：**Java / Kotlin（后端主力）**、**Go**（部分 Q-Commerce / Logistics 服务）、Python（数据 / ML 服务）、TypeScript + React（前端）、Kafka 重度、AWS + GCP 混合、Kubernetes、Postgres、DynamoDB、Redis。
- 团队方向：Consumer App、Restaurant Platform、Logistics & Dispatch、Q-Commerce (Dmart)、Payments & Fintech、Marketing Platform、AdTech、Global Discovery (Search / Recommendation / AI)、Developer Platform。
- 期待：能完成端到端微服务（设计 → 实现 → 测试 → 部署），对消息队列 / 高并发友好。

### 2.2 Data Analyst / Data Engineer / Data Science / ML Intern
- 技术栈：SQL（极重度，**几乎所有数据岗位面试都有 SQL**）、Python、Spark、Airflow、dbt、Looker / Tableau、BigQuery / Redshift / Snowflake、Databricks（部分团队）。
- 团队方向：Consumer Analytics、Restaurant Growth、Logistics Optimization、AdTech Analytics、Pricing & Promotions、Fraud & Risk、Recommendation / Search ML。
- 期待：强 SQL + 数据建模 + 商业 sense；ML 方向需要熟悉特征工程、A/B 实验、推荐系统。

### 2.3 特殊：Data Labeler Intern (多语种)
- 不需要工程背景；针对中文 / 土耳其语 / 阿拉伯语 / 西班牙语等 native speaker；为 Global Discovery（搜索 / 推荐 / NLP / 多语言）打标签。
- 性质：偏运营 / 数据准备，工资偏低，但门槛低、有本地工作经历。

## 3. 面试流程

### 3.1 通用流程（实习 + 早期工程）

1. **Recruiter Screen（30 min）** — 动机、与 DH leadership principles 的对齐、签证 / EU 工作权确认、薪资期望。
2. **Technical Interview I（60 min）**
   - SWE：项目深挖 + 1 道 LC Medium（数组 / 字符串 / hashmap），+ 简单 system design 概念（rate limiter / cache / pagination）。
   - DA / DE：SQL 现场（window functions、JOIN、aggregation）+ 业务案例。
3. **Coding / Pair Programming（60 min on coderpad.io）**
   - SWE：现场写代码，含 debugging、版本控制概念（git workflow）、环境搭建、learning agility。
   - 部分团队改为 take-home。
4. **Hiring Manager / Senior Engineer Round（60 min）**
   - 项目深挖、问"prioritize work、tackle problems、collaborate、empathy"。
   - 高级 system design：rate limiter、Kafka consumer 模型、缓存策略。
5. **Final / Cultural Fit**（部分流程）

整体周期：**2–4 周**，比 Zalando 快不少。

### 3.2 Tech Graduate Program（实习的姐妹项目，常被一起看）

- 时长：6 个月，含 upskilling sessions、社交活动、mentorship、12 周 capstone 项目。
- 流程：Recruiter → Engineer round 1（项目 + live coding）→ Engineer round 2（技术概念：版本控制、debugging、环境）→ Senior Engineer round（coding + 软技能）。
- 资格：CS / DE 或相关本/硕，最多 2 年工作经验，2 天/周 Berlin Mitte 办公室。

| 福利 | 内购餐券 / 公司食堂、健身房折扣、心理咨询、Pluxee benefits | 同上 + relocation 支持 + 培训预算 | 同上 |

## 5. 签证与跨境工作

- **EU/EEA 学生 / 居民**：自由工作；首选群体。
- **非欧盟在德国在读**：student visa 自带打工权（强制实习不算入 120/240 上限）。
- **海外申请者**：实习一般不 sponsor；但 Delivery Hero 在 Senior 岗位上较愿意 sponsor Blue Card（薪资门槛 2025 ~€45,300 / 年）。
- **全职毕业生**（TGP）：通常会 sponsor EU Blue Card，提供 relocation budget。
- **跨境**：Delivery Hero 在德国是注册雇主，无法在欧盟外远程办公；如果有意去 Talabat / foodpanda 本地团队，要分别走对应国家流程（Dubai / Singapore / Bangkok）。

## 6. 申请策略 & 时间线

| 阶段 | 时间 | 备注 |
| --- | --- | --- |
| Job posting | 全年滚动；TGP 春秋两批 | 主入口 careers.deliveryhero.com，部分岗位通过 LinkedIn |
| Application screening | 1 周 | Resume 关键词：Java / Kotlin / Go / Kafka / Microservices / SQL / Python |
| Recruiter call | 1–2 周 | 确认 EU 工作权、薪资期望 |
| Tech rounds | 1–3 周 | coderpad live coding 是主要形式 |
| HM + senior | 1–2 周 | 项目深挖 + 系统设计 |
| Offer | 平均 2–4 周 | 比 Zalando / Klarna 快 |

申请技巧：
- **海外候选人**：因为不 sponsor 实习签证，主要靠 TGP 或 Master Thesis 入口；Master Thesis 是绕过签证的好办法。
- **门槛较低但 volume 大**：建议同时投多个团队 / 多个品牌（Berlin 总部 + 本地 region），可申诉到不同 hiring manager。
- 准备 Kafka / Redis / 微服务相关问答，DH 工程问答很喜欢出 Kafka consumer 模型。
- Data Labeler 实习对非欧盟学生友好：语言能力 + Berlin 学生身份就够；可作为进入 Global Discovery 团队的踏脚石。
- 简历加分：在 fintech / e-commerce / 推荐 / 配送 / Q-Commerce 相关项目；Berlin 学校学生（TU Berlin、FU Berlin、HU Berlin、HTW、Hertie School、CODE）有 referral 优势。

## 8. 关键风险 / 提示

1. **不 sponsor 实习签证**：海外候选人需走 Master Thesis 或先在德国念书；本地学生友好度高。
2. **品牌差异**：Berlin 总部主要做平台 / 广告 / 推荐 / Q-Commerce；foodpanda Singapore、Talabat Dubai 等本地团队走的是各国本地流程，与 Berlin 流程不一样。
3. **门槛偏低 + 信号竞争**：因为 DH 在 Berlin 大量招实习，简历端"高质量但泛 e-commerce 项目" 反而比少数大厂实习更显得突出。
4. **薪资不高**：Berlin Praktikum 月薪 €1,200–2,000，扣除德国税与生活成本后所剩有限；Berlin 房租近两年快速上涨。
5. **Q-Commerce 风险**：Dmart 等 Q-Commerce 业务多次重组（2022 关停部分国家），实习项目可能因业务调整而被砍。
6. **Kafka / 微服务 / SQL 是高频考点**：实习面试也会考 Kafka 模型与 SQL 窗口函数；提前准备。
