# Canva (伦敦) 实习情报

> Canva 是澳洲设计独角兽 (估值 ~$32B),2024 年正式落地伦敦 Shoreditch,把伦敦设为欧洲总部。**伦敦办公室仍以商业、市场、设计为主,工程岗位有限**,绝大多数 Engineering Internship 仍集中在悉尼 ANZ。**信息相对匮乏**,以下信息以澳洲 SWE 实习面经为对照参考。

## 1. 公司速览

| 项目 | 内容 |
|---|---|
| 总部 | 悉尼 (澳洲) |
| 欧洲办公室 | 伦敦 Hoxton Square (Shoreditch),2023 / 2024 启动 |
| 伦敦员工规模 | ~250 人 (2024 数据,以非工程岗位为主) |
| 主营业务 | 在线设计平台 (200M+ MAU)、Magic Studio (生成式 AI)、Affinity 收购 |
| 技术栈 | TypeScript / React (前端)、Java / Go / Kotlin (后端)、AWS、Kafka、ML/Diffusion |
| 实习时长 | 12 周暑期 (ANZ 标准),伦敦实习是否单独开班尚不明确 |
| 申请窗口 | ANZ 入口约 3-5 月开放,12 月-2 月入职 |

## 2. 实习岗位与方向

> **重要提醒**: 截至 2025-2026 公开招聘页,Canva 几乎所有 Software Engineering Internship 均挂在 Sydney/Manila/Wuhan,**伦敦目前未见正式 SWE Internship 入口**。Canva 官方曾说明 Early Talent **不提供伦敦/欧洲 visa sponsorship**,需自带 work right。

| 岗位类别 | 招聘地 | 技术方向 |
|---|---|---|
| SWE Intern - Product Engineering | Sydney | Web 前端 / 编辑器渲染 |
| SWE Intern - Infrastructure | Sydney | K8s、Service Mesh、Observability |
| SWE Intern - Data Science / ML | Sydney | Recsys、Search、ML Platform |
| Graduate SWE (London) | London | 全栈,通过常规校招而非 internship 入口 |

## 3. 申请要求与签证

- **学历**: 倒数第二年 (penultimate year) 本科 / 研究生
- **Work Right (核心痛点)**:
  - ANZ Internship 仅接受 PR / 公民 / 学生签 → 海外学生若不在澳洲就读基本无门
  - 伦敦如开放 Graduate / Intern,Canva UK Ltd 是 Skilled Worker 持牌雇主,但 Early Talent 文案明示**不 sponsor**
  - 因此**国际学生若想伦敦实习,基本只能用 Student Visa 在英国就读的情况**
- **薪资**: ANZ Intern AU$80-95k pro-rated;伦敦 Grad SWE 预估 £55-65k base + RSU

## 4. 面试流程 (基于 ANZ Internship)

```
应聘 → 招聘官筛简历 (Resume + 短文 / 短题目)
     → HackerRank OA (1-2 题 LC Medium)
     → 视频面: 行为 15 min + 技术 45-50 min (LC + 代码 walkthrough)
     → Hiring Manager Behavioral + Project Deep Dive
     → 等待 ~3 周得 Offer
```

平均从首次接触到 Offer **68 天**,面试节奏比 FAANG 慢。

## 5. 面经摘录

| 时间 | 岗位/城市 | 流程概述 | 题目/题型 | 结果 | 来源 |
|---|---|---|---|---|---|
| 2024 | SWE Intern / Sydney | OA + 1 轮 panel | 一道 LC Medium (二维矩阵 / 字符串处理),行为问"为什么 Canva" | Offer | Glassdoor AU |
| 2024-Q1 | Internship Expression of Interest | 表单 + Resume Screen | / | Pool 候选 | GradConnection |
| 2023/24 Summer | SWE Intern Product Eng | 2 阶段 (行为 + 技术) | DSA 中等 + 系统设计 light | Offer | Whirlpool 论坛 |

> **伦敦实习面经在中英文论坛均未见;**信息匮乏**,本节以悉尼 SWE 信号代理。

## 6. 文化与口碑

- **正向**: 设计 + 工程互融,产品文化强;Slack 友好,扁平管理;伦敦办公室在 Shoreditch 创意区
- **挑战**: 工程岗高度集中在悉尼,伦敦目前更像销售 + 商务 hub;伦敦工程晋升通道与悉尼总部资源分配不对等
- **Glassdoor 评分**: 整体 4.2/5 (悉尼数据为主),Work-Life balance 评分高

## 7. 重点准备清单

1. **Canva 产品熟悉**: 试用 Magic Design / Magic Write,有自己用过的故事
2. **LC Medium 中频**: array / string / tree;不必死磕 hard
3. **行为问题**: STAR 模板;"为什么 Canva" 重点谈使命 ("empower the world to design")
4. **作品集 (前端方向)**: GitHub 上的小项目、设计感强的 UI,大幅加分
5. **思考产品**: 面试常问 "如何改进 Canva 的某个功能"

## 8. 总结评级

| 维度 | 评级 | 备注 |
|---|---|---|
| 信息丰富度 | 低 (信息匮乏) | 伦敦工程实习公开资料几乎为零,以 ANZ 信号代理 |
| 申请门槛 | 中 | 技术不难,但坑在 sponsor 政策 |
| Offer 转化预期 | 中-低 | 全球 100 个名额对 16-20k 申请 |
| 推荐指数 | 谨慎推荐 | 设计向同学加分大;纯后端方向收益有限 |

> **建议**: 若想冲 Canva 伦敦,目前更务实的路线是先冲 ANZ Sydney Internship (作为澳洲在读学生或暑期 globe-trotter),或直接申请伦敦 **Graduate SWE** 正职岗位,而非 internship。
