- **常见岗位名**: **Software Engineer (New Grad) — University Graduate** / **Software Engineer, IC1** / **SWE I**; Databricks 内部 level **IC1 = New Grad (0 yr)**, **IC2 = Junior (1-2 yr)**; 没有专门 "Graduate Programme" 品牌, 走标准 SWE + new-grad 蓄水池
- **欧洲地点**:
  - **Amsterdam (Zuidas)**: EMEA 最大工程 hub, New Grad 主要地点, 招 ~30-50 人/cohort (Mosaic AI / SQL / Unity Catalog / Delta Lake teams)
  - **Berlin (DE)**: 次大欧洲 hub, New Grad 名额开始增长 (2024-2025 起)
  - **Belgrade (RS)**: 工程 hub, New Grad 招本地 / 周边国家
  - **London (UK)**: 部分 GTM 加少量 Engineering New Grad
  - 与实习地点一致, 但 New Grad 比实习更倾向 Amsterdam (实习 Berlin 也接, New Grad 主推 NL)
- **是否有独立 Graduate Programme**: **No** — Databricks 没有 cohort-based grad scheme; 走标准 University Recruiting Pipeline (与 Google / Meta / Stripe 类似); 内部有 4 周 onboarding + bootcamp + mentor 制度
- **非EU签证 (全职)**:
  - Databricks Netherlands B.V. 是 IND **Recognised Sponsor**, Kennismigrant 经验成熟 (近 2-3 年快速建立)
  - Amsterdam New Grad SDE **完整 sponsor**: visa + relocation lump sum ($10K-$15K reported on Blind) + 60 天 corporate apartment + 单程机票 + 配偶协助
  - **30% ruling**: New Grad base **远超** 2026 年 < 30 岁门槛 (€46,107), 100% 适用; 与实习不适用是数量级差异
  - **对国际生 friendly 度 EMEA 顶级** — 与 Stripe / Snowflake 同档, Databricks 是少数从北美 / 印度 / 中国 top 校园直接 sponsor Amsterdam New Grad 的公司
- **薪资范围 (base, gross/年)**:
  - **Amsterdam New Grad SDE (IC1)**: **€75K-€90K base + €40K-€60K RSU/年 (4-yr vest, 40-30-20-10 schedule) + ~10% target bonus** ≈ **€125K-€155K total** (Blind + Levels.fyi 推算, 比 US IC1 $250K 低 ~40%)
  - **Berlin New Grad SDE**: 与 Amsterdam 接近, ~€75K-€88K base + RSU

- **录取竞争**: Amsterdam New Grad SDE 录取率 **~1-2%** (申请池 5,000-8,000/cohort), 是欧洲 entry-level 最难入门之一 (与 Stripe / Citadel / Two Sigma 同档); **国际生 friendliness 高于 Booking / Adyen** (Databricks 全球 hire 文化, 从北美校园直推欧洲 office)

## 2. 签证与国际学生政策

- Databricks NL 是 IND **recognised sponsor**, 全职 SWE 转正全额支持 Kennismigrant + 30% ruling
- **实习生**:
  - EU/EEA → 直接申请, 无需签证
  - 已在 NL 院校就读非 EU → 学习居留 + Nuffic 实习协议 (Databricks 经常用)
  - 不在 EU 院校的非 EU → 需 GVVA, Databricks **支持但优先长期 thesis intern**; 12 周短期暑期对非 EU 海外学生 sponsor 意愿较低
- **Relocation**: Amsterdam intern offer 通常含 **单程机票 + corporate apartment / housing stipend** (12 周项目设计就是 self-contained); 具体数字未公开
- **30% ruling**: 实习生不适用; 全职转正 Amsterdam SWE I 起薪通常超门槛, 适用

## 3. 面试流程

Databricks 流程**长且严**, 是榜单中**接受率最低**的公司之一 (与 Stripe / Citadel / 顶级 quant 同级)。

### Amsterdam SE Intern 标准流程 (5–7 周, 4–5 轮)
1. **HR / Recruiter Call** (30 分钟)
   - 经历 + 动机 + "Why Databricks" (**必答**, 期待具体 — Spark? Delta? MLflow? Mosaic?)
   - 项目深入, 资历核查
2. **Karat Phone Screen** (60 分钟, **外包给 Karat 平台**)
   - 通常 **2 题 (1 easy + 1 medium)**, 部分报告会出 3 题但时间不够
   - **不能用 AI / Google** (Karat 是带录像的真人监考)
   - 题型: 数组 / hashmap / linked list / tree, **干净代码 + 测试用例思维**优先于通过 cases
3. **第二轮技术算法** (60 分钟, virtual, 现场写代码 + 高级 + 实习生联合面)
   - LeetCode medium 范围, 偏 graph / DP / sliding window
   - **必带 follow-up**: 优化时间空间, 写测试, 讨论 edge case
   - 部分组会问基础知识 (hash table 内部实现, generic, OOP 概念)
4. **Behavioral / Hiring Manager Round** (45 分钟)
   - "Why Databricks?" "Tell me about Databricks values"
   - 经典: "What negative feedback have you received?" (一份 2025-10 报告原文)
   - 项目深入 + 自我反思 + culture fit
5. **(偶有) 第二轮 manager / 额外技术** — 视组而定, intern 不总有

### Full-time / Senior 对比 (供参考)
- 全职流程: HR + 2 phone screens + **5 onsite rounds** (~9–10 小时), 比 intern 长不少
- onsite 含: 2 coding (LeetCode medium/hard), 1 system design, 1 deep dive (Spark/distributed), 1 behavioral
- 反馈: "比 Facebook / Google 更难, hiring manager 明确说 'looking for the best'"

### 题型重点
- **算法**: LeetCode medium 为底, 但要求**写得干净 + 一次过 + 自带测试**
- **真实考过**: Topological sort for job graph (2025-10 报告), LeetCode 经典图题
- **System Design (intern 偶有, 全职必有)**:
  - 设计一个简化版 Spark scheduler
  - 设计 distributed file system metadata
  - 设计 Delta Lake 概念 (transaction log + parquet)
- **Spark/Distributed 内部 (后期轮次)**:
  - Spark shuffle 原理, partitioning, broadcast vs shuffle join
  - RDD vs DataFrame vs Dataset
  - 容错: lineage, checkpoint
- **行为面 (Databricks 8 Behaviors)**: ownership, customer obsession, take risks, raise the bar

## 7. 踩坑与社区评价

### 警惕点
- **流程极严**: 5–6 轮, 接受率 << 5%, Blind 上多帖抱怨"比 Facebook/Google 都难"
- **Long loop**: full-time onsite 9–10 小时, intern loop 也 5–6 小时虚拟; 时间投入巨大
- **picky 文化**: hiring manager 拒人风格"looking for the best candidate", 没有 mediocre offer
- **题目难度上升**: 2024 后 medium 居多, follow-up 要写测试 + 性能优化 + edge case, 不只是 "AC 即可"
- **不能用 AI 工具**: Karat 监考严格, ChatGPT / Copilot 帮助不到
- **Amsterdam intern 名额少**: 估计每年 < 20 人 cohort (vs 北美几百人)
- **pre-IPO 估值变动**: RSU 价值不确定, 早期员工享受到 IPO upside, intern 享受到的程度依赖入职时间

### 优点
- **品牌顶级**: Databricks 简历背书 = 顶级 AI/data 公司, 跳 Snowflake / Confluent / OpenAI / Anthropic 都加分
- **stipend 极高**: 本榜单最高之一, Amsterdam 实习足够过得很好
- **mentorship 强**: 12 周有明确 project + mentor + 同期 cohort 活动
- **return offer 率高**: 公开报告显示 return rate **70%+** (含 Amsterdam)
- **技术栈 cutting edge**: Spark, MLflow, Mosaic AI (LLM training/serving), Unity Catalog
- **Amsterdam Zuidas 位置**: 现代化办公区, 公共交通方便

### 利与弊
- (+) 顶级技术 + 顶级 stipend + 顶级品牌
- (+) return offer 率高, intern → full-time → senior 路径明确
- (-) 流程难度极高, 准备时间投入大
- (-) Amsterdam 名额少, 非 EU 学生概率更低
- (-) IPO 未发生前 RSU 价值不确定
