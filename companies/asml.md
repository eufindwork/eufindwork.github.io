> 更新时间: 2026-05-26

> 公司背景: 全球极紫外光刻机 (EUV) 唯一供应商, 台积电/三星/Intel 的核心设备供应商. HQ 在 Veldhoven (Eindhoven 旁), 全球员工 ~40,000 人

## 1. 基本信息

- **公司全称**: ASML Holding N.V.
- **HQ**: De Run 6501, Veldhoven (Eindhoven 都市圈, 距 Eindhoven 中央火车站 ~15 分钟车程)
- **荷兰其他位置**: 主要 Veldhoven, 部分团队在 Eindhoven, Helmond
- **海外**: San Diego, San Jose, Wilton (CT), Chandler (AZ), Hillsboro (OR), Linkou (台湾), Pyeongtaek (韩国) 等
- **业务**: 光刻系统 (DUV / EUV / High-NA EUV), 计算光刻软件, metrology, service
- **股票**: ASML (Nasdaq) / ASML.AS (Amsterdam), 2024 年市值 ~€280B
- **实习岗位类型**:
  - **Software (SW) Internship**: 桌面 / server / embedded C++, Python, machine learning infra
  - **Embedded Software Internship**: 板级 C/C++, real-time, RTOS
  - **Electrical Engineering (EE) Internship**: 电路设计, signal integrity, power
  - **Computational Lithography Internship**: ML for resist modeling, OPC, mask synthesis (**热门, 高门槛**)
  - **Physics / Optics Internship**: 实验光学系统, electron optics modeling
  - **Mathematics Internship**: 优化, 数值方法, 编程改进
  - Mechanical / Mechatronics / Process / Control Systems 等其他工程实习
- **实习时长**: **6–9 个月** (官方推荐最少 6 个月, 9 个月最佳, thesis 路径可达 12 个月)
- **每周工时**: 4–5 天 (其中至少 2 天 on-site, 视项目)
- **申请窗口**: rolling, 但**建议至少提前 3 个月**申请 (含 4 周 pre-onboarding)

> 区别于实习: 实习 6-9 个月走 GVVA 或 Nuffic 协议; **Starter (新毕业生)** 全职合同, 走 Highly Skilled Migrant + 30% ruling, package 数量级跃升

- **常见岗位名**: **Starter Software Engineer** / **Starter Design Engineer** / **Starter Mechatronics Engineer** / **Junior Computational Lithography Engineer** / Starter Process Engineer; ASML 内部 "Starter" 是 0-1 yr 经验全职官方标签
- **欧洲地点**: **Veldhoven (HQ, De Run 6501)** 占 70%+ Starter 名额; Eindhoven city 部分团队; Helmond (制造), Berlin (DE, 部分软件 / 算法), Linkou (台湾) — 欧洲新 grad 几乎全部 Veldhoven
- **是否有独立 Graduate Programme**: **Yes — 多个并行 program** (非单一 cohort):
  - **Technology Development Program (TDP)** — 2 年 rotational, 跨 R&D / Manufacturing / Customer Support, 招 Master / PhD, 是 ASML 最 prestigious 的 grad track
  - **Engineering Career Starter** — 直接全职 SWE / EE / Optics, 不轮岗
  - **Customer Support Engineer Starter** — 派驻 ASML 客户 (TSMC / Samsung / Intel), 含海外派遣
  - **PhD-track Graduate** — 与 TU/e, TU Delft, Twente 合作
- **非EU签证 (全职)**:
  - ASML 是 IND **顶级 Recognised Sponsor** (全 NL 第二大 sponsor, 仅次于 Philips), 每年 sponsor 2000+ Kennismigrant; 流程 2-4 周 (远比社会面快)
  - 提供 **full relocation**: 单程机票 + 60 天 corporate apartment + 永久 BSN/银行/医保协助 + 配偶 search agency + 国际学校协助 (适合带家属)
  - **30% ruling**: Starter SWE 起薪通常**超过 €46,107 (< 30 岁) 门槛, 100% 适用**; 与实习仅拿 €450 housing allowance 相比是数量级差异
  - **EU Blue Card / Highly Skilled Migrant 双路径**: ASML 默认走 Kennismigrant (NL 国内最快), 但若有跨欧盟流动需求会改 Blue Card
- **薪资范围 (base, gross/年)**:
  - **Veldhoven Starter SWE (L6, entry)**: **€82.4K base + €9.6K bonus ≈ €92K total** (Levels.fyi 2026-05); 加 13 月薪 (8% holiday allowance)
  - **TDP Starter**: 略高于普通 Starter, ~€85K-€90K base + 更高 bonus pool
  - **PhD-track Graduate**: €95K-€110K base 起 (Master + PhD 折抵)
  - **Stock**: ASML.AS RSU, Starter level 通常**没有 RSU**, L7+ 才开始 grant; bonus 与公司业绩强挂钩 (2023 历史 ~15-20% 高峰, 2024-2025 因 EUV 周期回调到 10-12%)
  - **30% ruling 后实得**: €82.4K base → ~€68K taxable → 实际到手约 €5.0K-€5.5K/月 net (远高于荷兰本地同龄人)

- **申请窗口**: **rolling year-round**, 没有固定 cohort 截止; Career portal 持续 ~200+ Starter 岗位开放; 建议毕业前 6 个月开始投; 实习转 Starter 转化率非常高 (公开 ~30-40%)
- **面试流程差异 vs 实习**:
  - 流程结构相似 (CV → Recruiter → Hiring Manager → Technical → Offer), 但**技术轮加深** — 多一轮 **technical panel (3 名 Senior + 1 名 Architect)**, 现场 whiteboard 系统设计

  - 整体 4-6 周, 比实习 (4 周) 略长
- **录取竞争**: ASML Starter 在 NL 最有竞争力的 employer brand 之一 (Universum 排名 NL #1 Engineering employer), 申请池每年 30,000+, Starter 录取率 ~5-8%; **对国际生极 friendly** (公司本身 40% 员工是国际人才); 中国学生通过率高于 booking / adyen 因技术 fit > 业务 sense

## 2. 签证与国际学生政策

- **EU/EEA 学生**: 无需签证, 实习直接走
- **非 EU 学生**:
  - 已在 NL 院校就读 → 学习居留 + Nuffic 实习协议 (ASML 经常用)
  - 不在 NL 院校 (其他欧洲国家或中国/美国院校) → **GVVA (combined permit residence+work for internship)**; ASML 是 IND recognised sponsor, 可办, 处理周期约 2–6 周
- **关键差异**: ASML 比 SWE 互联网公司**更愿意为长期实习 (6–9 个月) 办签证**, 因为 ROI 高 (短期 SWE 实习公司一般不愿走 GVVA 流程)
- **住房补贴 (Housing Allowance)**:
  - **EU 学生**: €450/月, 条件: 原居住地距 Veldhoven ≥ 80 km, 并搬到距 ASML ≤ 30 km 新住址, 自己找房 + 提供租赁合同
  - **非 EU 学生**: 官方页面**未明确**, 1point3acres 多份报告显示非 EU 学生**通常也能拿到类似补贴**, 但需 case-by-case 与 HR 确认
- **30% ruling**: 实习生**不适用**; 全职转正若符合 highly skilled migrant 标准, 30% ruling 适用 (ASML 是大量 highly skilled migrant 雇主)

## 3. 面试流程

ASML 流程相对**比互联网公司轻**, 重 fit + project match, 少 leetcode hard 题。

### 标准流程 (4–6 周, 4 步)
1. **CV + Motivation Letter 投递**
   - **关键**: motivation letter 比 SWE 公司更重要, ASML 极看重"为什么 ASML / 为什么这个项目"
   - 7 个工作日内通知是否进入下一轮
2. **HireVue 视频面试** (Pre-recorded, async)
   - 几道预设题, 单题 60–120 秒回答, 大多关于动机/经验/团队合作
   - **几乎没有技术题**
3. **第一轮 1v1 (Recruiter / HR)** — 视频, 30 分钟, 动机 + ambition + culture fit
4. **第二轮技术面 (with team / supervisor)** — **on-site 强烈推荐** (Veldhoven 园区, 含园区参观)
   - 项目 deep dive (resume / thesis / 过往项目)
   - 技术问题: C/C++ 基础, link list, OOP, 并发, 偶有 system 概念
   - 对 Comp Lithography / ML 岗: numpy / pytorch + 光刻 fundamentals + 论文讨论
   - EE 岗: 电路分析 + signal integrity + 实验经验
   - 偶有"白板写代码" (实习一般为简单题, full-time 较多 medium)

### 按岗位差异
- **SW (general)**: 偏 C++/Python, OOP, 多线程, debugging; 系统软件背景加分
- **Embedded SW**: C/C++ pointers, memory layout, RTOS, ISR, 偶有 hardware/software interface 题
- **EE**: 电路 / FPGA / signal integrity; 实验室经验权重高
- **Computational Lithography (CompLitho)**:
  - **强烈推荐 PhD / Master + 较强物理或数学背景**
  - 题目: 偏 lithography (OPC, source mask optimization), 数值方法 (FFT, FDTD), ML for physics (NN as surrogate)
  - 编程: Python + numpy/scipy, 偶有 C++ 性能优化讨论
- **Physics/Optics**: 经典波动光学, beam propagation, electron optics, Monte Carlo

## 7. 踩坑与社区评价

### 警惕点
- **Stipend 低**: 这是 ASML 实习最被诟病的一点, 1 星 (在 JoinTaro)
- **Veldhoven 地理位置**: 不是 Amsterdam! 没车几乎活不下来 (公交可达但很慢, 每天 30–60 分钟通勤是日常); Eindhoven 城市规模较小, 夜生活 / 国际化程度远不及 Amsterdam
- **Monolithic codebase**: 实习生反馈"代码库巨大且复杂, 头几个月学习曲线陡"
- **Onboarding 时间长**: 4 周 pre-onboarding, 含 security 培训, 部分实习生头 1 个月做不出实际工作
- **保密**: EUV / High-NA 是国家级技术, NDA 严格, 简历上能写的内容受限
- 部分实习生反馈"过于 hierarchical / 老派"; ASML 是欧洲制造业大厂气质, 不是硅谷快节奏

### 优点
- **职位丰富**: SW/EE/Phys/Mech/Optics/ML 全覆盖, 几乎所有 STEM 背景都能找到匹配
- **mentorship 制度成熟**: 实习生反馈"resp 极快, 同事帮助多"
- **转正稳定**: 实习→全职 conversion rate 高, 全职待遇 + 30% ruling 后包很有竞争力
- **简历背书顶级**: ASML 是半导体行业绝对 top tier, 后续跳 Intel / TSMC / Nvidia / Samsung / Synopsys 都吃香
- **长期实习**: 6–9 个月足够做出有意义的 thesis / project

### 利与弊
- (+) 进入半导体行业最稳的实习路径
- (+) Visa 支持成熟, 9 个月长度让 GVVA 流程值得办
- (-) Stipend 几乎是榜单最低
- (-) 地理位置非 Amsterdam, 城市生活体验弱
- (-) 文化偏传统制造, 不适合追求"硅谷 startup 氛围"的人
