## 3. 面试流程

| 轮次 | 内容 | 时长 | 备注 |
|------|------|------|------|
| 1. Online Assessment | **SIG Quantitative Evaluation (SQE)**: 17 道 math + probability | ~60 min | 含 linear system, expected value, grid path-finding; "20 min" 短版用于 prescreen |
| 2. HR Phone Interview | vibe check + 3 道 probability / EV 题 | 45 min | 不是纯 BQ, 有量化筛 |
| 3. Trader Interview (Round 1) | 1 名 trader, analytical + behavioral + brain teaser (decision-making) | 60 min | poker / cards / dice / coins; 重 EV 计算 |
| 4. Take-Home Project | 48h, 任何语言, 分析 trade sequence + propose strategy | 48 hours | SIG 独有, vs Optiver/IMC 没有 |
| 5. Final Onsite (Dublin) | trader + HR "good cop bad cop", heavy EV + MM game based on estimated fair value | 半天 to 1 天 | 飞机 + 酒店 SIG 出 |
| 总流程 | **2-3 个月, 5 轮** | very slow process | WSO + Tradermath 一致评价 |

**SIG-specific 说明**:
- **Decision theory + Poker**: SIG 最有名的是面试中"假设你打 poker, ..."系列题 — 候选人需要懂 expected value + Kelly criterion + Bayesian update
- **48h take-home** 是 quant 圈罕见: 一段历史 trade 数据 + 让你写一个 strategy + report; SIG 用此评估"边做边学"能力
- **SWE 轨** OA 偏 algorithm (HackerRank / Codility) + 系统设计, vs Trader 轨 SQE 偏 brain teaser
- **流程慢是公开痛点**: Glassdoor + WSO 多帖抱怨"2-3 个月 wait", 建议同时投 Optiver + IMC 平行

## 7. 踩坑与社区评价

- **流程慢 (2-3 个月)**: 业内最长流程之一; WSO + Glassdoor 大量 "wait time too long" 抱怨; 建议平行投 Optiver + IMC
- **48h take-home 是 entry pain point**: 48 小时高强度, 在校学生 + 全职面试碰巧期间困难
- **Decision theory 重**: 没读过 Thinking in Bets / Kelly criterion / Bayesian 直接吃亏
- **Dublin 生活成本极高**: 中心 1 房 €2,000+/月, grad base €70-100K 实际可支配低于 Amsterdam Optiver (即使 nominally TC 较高)
- **没有 30% ruling 等效**: 爱尔兰个税相当于 NL 不享受 ruling 的水平 → 实际税后差距 Amsterdam Optiver/IMC + 30% ruling 可能反超 Dublin SIG
- **私有公司无 RSU**: 全 cash + discretionary bonus; bonus 波动 PnL 敏感 (2022 减 42% per head 公开报道) [来源 (搜索结果): https://www.efinancialcareers.com/news/a-hft-firm-that-paid-1-1m-per-head-in-2022-saw-pay-fall-42]
- **Dublin 中国 / 印度生友好度**: ⭐⭐⭐⭐ — CSEP 流程比 NL Kennismigrant 略宽松, 但 Dublin 文化"East Coast US 化"程度高
- **Glassdoor 4.1/5**: 与 Flow Traders 持平; 抱怨集中在"工作强度高 (60-70h/week 常态)" + "面试流程慢"
