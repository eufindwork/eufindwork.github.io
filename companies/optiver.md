## New Grad / Junior 岗位

> 信息丰富度: ⭐⭐⭐⭐⭐ (Levels.fyi L3 €185K total 数字直读 + 官方 Graduate SWE / FPGA 岗位实时开放 + 1point3acres 80in8 + ZapN 二轮实测)

- **常见岗位名 (区分关键)**:
  - **Graduate Software Engineer** (主, C++ 方向) — Trading Systems / Low-latency Infra
  - **Graduate FPGA Engineer** / **Junior FPGA Engineer** — Tech 部门 (2026-05 Amsterdam 实时开放)
  - **Quantitative Researcher (QR)** — research / pricing model
  - **Quantitative Trader** — 真 trader 角色, 不在 SWE 路径, 走单独 trading 招聘 + 自己的 Trading Bootcamp
  - **Trading Tech Engineer** — 介于 SWE 和 trader 之间, support trader 现场
- **欧洲地点**: Amsterdam (主, EU HQ) / London (二线, options + ETF) / 北美 Chicago + Austin / APAC Sydney + Shanghai + Taipei
- **是否有独立 Graduate Programme**: **Yes** — **Optiver Graduate Programme** 含 3 轨 (Tech / Trading / Research), Tech 轨 6-12 周 onboarding bootcamp + rotating team placement, cohort 规模 EU ~30-50 人 (不公开精确数字)
- **非EU签证 (全职)**:
  - **NL Kennismigrant (HSM) 2026**: <30 €4,357/月 (€52,284/年 gross), ≥30 €5,942/月 (€71,304/年 gross), 硕士 reduced €2,801/月 (€33,612/年) — Optiver Graduate SWE base €100K+ 远超所有门槛, **100% 自动符合**
  - **30% ruling**: 适用, 前 5 年最多 30% 免税 (2024-01 法改: 5 年内逐档下调到 30%/20%/10%, 仍是最强 EU 财务优势之一)
  - **Visa 实际**: Optiver 官方"sponsors HSM (Highly Skilled Migrant/Kennismigrant) visas for software engineers working on low-latency trading systems, infrastructure, and FPGA development" — IND 流程 2 周, 已为 H-1B 失败者主流欧洲安置路径 [来源 (实际打开过): https://nextleveljobs.eu/blog/visa-sponsorship/nl]
  - **Intern 签证**: case-by-case, "is able to support visa sponsorship on a case by case basis, which is discussed during the interview process" — 不是默认 yes
- **薪资结构 (量化特有, 必填)**:
  - **Intern (SWE, Amsterdam)**:
    - Levels.fyi 2025 Amsterdam **$126.14/hour ≈ €115/小时, 10 周 ≈ €46K stipend equivalent** (远高于 Big Tech 实习); 业内常说月薪 €5,000-6,000 量级
    - **Housing 全包** (中心区豪华公寓), **机票全包**, 通勤补贴, 早晚餐 + 健身房, BSN / bank account 办理协助
    - 类比: Chicago Summer 2026 intern $175/hour + $10K sign-on; Amsterdam 略低于美国但日均工时更短
  - **全职 New Grad (Amsterdam Graduate SWE, L3)**:
    - Levels.fyi 2026-05-28 直读: **L3 total €185K** = base $118K (≈€109K) + bonus $91.5K (≈€85K) + stock $6K (RSU 罕见, 4 年 vest)
    - L4 total **€217K** (base ≈€81K + bonus ≈€120K), L5 **€313K** (base ≈€81K + bonus ≈€255K) — **bonus 占比极高 (50-80%)** 是 Optiver 的招牌
    - 6figr.com 数据: 全球 Optiver SWE $300K-$686K 区间 (含资深 + 美国 office)
    - Sign-on: $10K-$15K (intern conversion 路径常见)
  - **全职 New Grad (Trading 轨, Amsterdam)**: 公开数据少, Glassdoor 显示 entry trader €70-100K base + 巨型 discretionary bonus (Y1 总包传闻 €150-300K, 与 desk PnL 直接挂钩)
- **申请窗口**:
  - **Graduate SWE / FPGA**: 2026-27 cycle 滚动开放 (rolling), 欧洲专属 deadline 偏向 Oct-Dec 年末
  - **Internship 2026**: Amsterdam 现在锁 2027 cohort (penultimate year 学生, 2027 毕业), Sydney / Chicago / Austin 已开 2026 Summer
  - **2026-05 实时开放 (Amsterdam)**: Junior FPGA Engineer, Graduate FPGA Engineer (Tech 部门, 通过官方 careers 板) [来源 (实际打开过): https://optiver.com/working-at-optiver/career-opportunities/]
- **面试流程差异 vs 实习**:
  - **Optiver mental math (80in8 → 当前 50 题 / 8 分钟)** — quant 圈最著名筛选 — 必写: 80 道四则运算 (整数 + 分数 + 小数), 8 分钟, 每对 +1 错 -2, 不可跳 / 不可改, 及格 ~55, 竞争力 70+; **2026 SWE / Quant 申请均强制此轮**

  - Trader 走 mental math + market making game + brain teaser
  - 全职多一轮 Onsite "Super Day" (Amsterdam HQ, 飞过去 2 天, 含 trader + tech + culture)

## 3. 面试流程

| 轮次 | 内容 | 时长 | 备注 |
|------|------|------|------|
| 0. OA 1 (mental math) | **80in8 (实际 50 题 / 8 分钟)** — 整数 / 小数 / 分数四则运算 | 8 min | 不可跳, 不可改; 错 -2 对 +1; 及格 55+, 竞争力 70+ |
| 1. OA 2 (SWE) | HackerRank 2 道编程 (1 OOP 系统设计 + 1 算法), C++/Java/Python | 120 min | 1point3acres 2026 batch 报告系统设计是 "news subscription system" |
| 1. OA 2 (QR) | HackerRank 1-2 道概率 + sequence | 75-90 min | 报告"问题描述极长 + 难度炸裂"; QR 2026 OA 含 sequence 题 |
| 1.5. ZapN | 9 个神经反应小游戏 (反应速度 / 模式识别) | ~30 min | 2026 新增二轮筛选, 速度敏感型 |
| 2. 电面 / Coderpad | 1 名 senior engineer, live coding (circular buffer / LRU 变种 / queue) | 60 min | 重沟通 + 边界 case, 不要求最优 LeetCode 解 |
| 3. 技术二面 | 系统设计 (exchange / order book / FIX gateway), C++ memory model | 60 min | 仅 SWE; QR 走 stats / probability deep dive |
| 4. Super Day (Onsite Amsterdam) | 2 天 — 2 轮技术 + culture fit + lunch + office tour | 2 days | 飞机 + 酒店 Optiver 出; 一般 1 周内 offer / reject |

**Optiver-specific 说明**:
- mental math 是**绝对淘汰门槛** — 即使 SWE 岗也必须过, 不会因为 "我是写代码的不是 trader" 而豁免
- 官方建议: zetamac / arithmetic.zetamac.com 每天练; 25-30 分一天目标到 40-50 分 / 2 min
- 2026 报告: "oa 难度炸裂" "Optiver 2026 QR Intern OA" 多篇 1point3acres 帖子 [来源 (搜索结果): https://www.1point3acres.com/bbs/thread-1145878-1-1.html]

## 6. 申请渠道与建议

- **官网投递**: https://optiver.com/working-at-optiver/career-opportunities/ — Workday 后台, EU 申请用 Amsterdam location filter; FPGA / Tech / Research / Trading 分开列
- **内推**: 重要 — 内部员工 referral 直接跳过简历筛 (intern OA 通过率本就低); 校园大使 (Cambridge / Oxford / TUDelft / TU Eindhoven / ETH / EPFL) 是主要 sourcing 渠道
- **CV 重点**:
  - **量化必修**: Heard on the Street + A Practical Guide to Quantitative Finance Interviews (绿皮书 Xinfeng Zhou) — Optiver MM game 几乎所有题型在内
  - **算法竞赛**: Codeforces ≥ Expert (1600+) / IOI / ACM-ICPC Regional 直接对 Tech 加分; IMO / Putnam 对 QR / Trader 必杀
  - **C++ 深度**: cppreference 内存模型 / lock-free queue / SPSC ring buffer 一定要懂
  - **学校梯队**: TU Delft / TU Eindhoven / Tilburg / Bocconi / TUM / ETH / Imperial / Cambridge / Oxford / Warwick (UK) — Optiver 校园活动主点
- **同档备选**: IMC Trading (Amsterdam, 同梯队, 流程略短) / Flow Traders (Amsterdam, ETF MM 偏 niche) / Jane Street (London, 美国系一档) / Citadel Securities (London/NY) / HRT (London/NY)

## 7. 踩坑与社区评价

- **80in8 / 50in8 是绝对淘汰门槛**: WSO 历史帖子 "Optiver 80 in 8 numerical test seems impossible. Is it even possible to pass this test without luck?" — 答案是练; zetamac 60+ 是入场券, 不练直接挂 [来源 (搜索结果): https://www.wallstreetoasis.com/forum/trading/optiver-80-in-8-numerical-test-seems-impossible-is-it-even-possible-to-pass-this-test]
- **ZapN (2026 新增) 反应速度敏感**: 9 个小游戏 + 75 分钟极限, 1point3acres 用户报告"反应慢直接挂" [来源 (搜索结果): https://www.1point3acres.com/bbs/thread-1144268-1-1.html]
- **bonus 不稳定**: 公司 PnL 差年份 (2018 / 2022 部分 desk) 报告 bonus 缩水 30-50%; Y1 base 只占 30-40%, 谈薪不能只盯 base
- **文化偏"工程师 + flat"**: 没有 dress code, 没有 cubicle, 但工作强度高 — 周末加班不强制但常见; 国际生反馈"招聘流程公平, 不歧视非欧本科"
- **Glassdoor 4.4/5 + 高满意度**: 但 30% ruling 政策变动是 2024 后入职者的最大担忧
- **国际学生友好度**: ⭐⭐⭐⭐⭐ — Amsterdam 是 EU 最国际化办公室之一, IND HSM 流程 2 周, 不需要荷兰语
