## 2. 签证与国际学生政策

- **Paris 实习**: 法国 convention de stage 三方协议, 学校 + 公司 + 学生; 非欧学生需有法国/欧盟学籍; 实习津贴 (gratification) 法定 €659/月起, Squarepoint 给 €3-4k/月
- **London 实习**: Tier 4 Student Visa, vacation full-time OK; 不办独立 work permit
- **Paris 全职**: Passeport Talent Salarié Qualifié = €39,582/年 (2026, 新定额, 不再 SMIC 倍数); Squarepoint Paris QD base €70K+ 远超 → 合规
- **London 全职**: Skilled Worker £41,700 (2026); Squarepoint UK QD £85.5K+ → 合规; Squarepoint UK Ltd 有 Tier 2 sponsor license
- **法语要求**: Paris office 工作语言英语 + 法语 mix (技术内部多英语, 行政 / HR 有法语); 法语 B2 加分但非门槛
- **路径推荐**: 非欧学生若同时拿 Paris + London offer, Paris 走 Passeport Talent 4 年 multi-year permit 是欧洲最长, UK Skilled Worker 5 年 ILR 是终点更近 — 视长期规划选

## 3. 面试流程

| 轮次 | 内容 | 时长 | 备注 |
|---|---|---|---|
| 0. CV screen | recruiter call 介绍 role | 30 min | 极快, 1-3 天回复 |
| 1. OA | Python 重: data structures + probability + stats | 90 min | HackerRank, 全 Python (即使 QD 也不强制 C++) |
| 2. Tech Phone 1 | algorithm + math/finance + Python 深问 | 60 min | LeetCode med, 约 50% 是纯 algo |
| 3. Tech Phone 2 | system design lite + KDB+/q (若 QD) / stats + signal design (若 QR) | 60-75 min | KDB+/q 知识加分大但非必须 |
| 4. Senior Quant/Dev 对话 | problem-solving depth + cultural fit | 60 min | "conversational with senior" |
| 5. Final / Onsite | 4-5 个员工 mix, 含 head-of-team / hiring manager | 半天 | London 或 Paris, 看 base location |
| 6. Offer | 1-2 周内 | — | 平均 31 天全流程 |

## 7. 踩坑与社区评价

- **流程相对快**: 平均 31 天 (比 QRT 4-8 周 / Akuna 数月快)
- **Python heavy OA**: 即使 QD/QR 路径也是 Python OA, 不是 C++; 但全职 QD 会写很多 KDB+/q + C++
- **Glassdoor 39.8% positive (中等偏低)**: 主要抱怨是 (1) recruiter follow up 慢 (2) final round 决定不透明 (3) bonus 不透明 — 法国系基金通病
- **法语非必须**: Paris 团队工作语言英文为主, 法语 B2 是 nice-to-have
- **国际学生友好度**: ⭐⭐⭐⭐ (Paris + London + Madrid + Zug + Geneva 多站点 → visa 选择多)
- **vs QRT**: Squarepoint 更老 (2014 vs QRT 2018) + 规模略小但 AUM 接近 + 文化更 "low-key academic"; QRT 更扩张 hire / 工程师占比相对低 / deferred comp 更重
