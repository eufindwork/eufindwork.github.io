# Zalando 实习情报（Berlin / Helsinki / Dublin）

> 范围：Software Engineering Intern、Machine Learning / Applied Science Intern、Data Engineering / Data Science Intern。Berlin 是 Zalando 最大的 tech hub，Helsinki 是第二大 tech hub（专门有 Future Talent Program），Dublin 也有 fashion store engineering 团队。

> 区别于实习: 全职 entry-level (0-2 年经验); Zalando 内部用 C-band (C1-C7), 应届通常落 **C5 (Junior Software Engineer)** 或 **C6 (Software Engineer)**

- **常见岗位名**: Junior Software Engineer (C5)、Software Engineer (C6)、Junior Data Engineer / Data Scientist / Applied Scientist、Frontend Engineer Junior; 招聘页明确按 C-band 命名, "Junior" 标签清晰
- **欧洲地点**: 与实习一致, 全职覆盖更广 — **Berlin (HQ, 最大 tech hub)、Helsinki (第二 tech hub)、Dublin (Fashion Store engineering)、Lisbon (Logistics 偏多)、Madrid (近年新增)**; Junior SWE 主要在 Berlin / Helsinki
- **是否有独立 Graduate Programme**: **No formal cohort program**, 但有两条结构化通道:
  - **Helsinki Future Talent Program** (源自实习, 部分转 full-time Junior SWE 进入 C5)
  - **Master Thesis → C5 Junior** 通道 (Berlin / Helsinki 都常见)
  - 直接 Junior Engineer (C5) req: rolling 全年, 没有 cohort onboarding, 走 standard onboarding 流程
- **非EU签证 (全职)**:
  - **EU Blue Card (Germany)**: 2025 阈值 ~€45,300/年 (紧缺职业 ~€41,041), Zalando C5 起薪通常达标; Zalando **常规 sponsor 全职 Blue Card** (与实习的"不 sponsor"形成明显对比), 提供 €6,000 relocation budget + 首月 corporate housing
  - **Finland**: Helsinki 通过 Finnish residence permit for employed person, 类似 Blue Card 流程
  - **Ireland (Dublin)**: Critical Skills Employment Permit (薪资 ≥ €38,000), Zalando Dublin 偶有 sponsor
- **薪资范围 (base, gross/年)** (Levels.fyi / Glassdoor 2024-2025):
  - **Berlin C5 Junior SWE**: base ~€55,000-65,000, +bonus 5-10%, TC ~€58k-72k

## 2. 岗位画像

### 2.1 Software Engineering Intern
- 技术栈：Java（主力）、Kotlin、Scala（部分历史团队）、Python、Node.js / TypeScript、React、AWS（重度，几乎全在 AWS）、Kafka、Postgres、DynamoDB、Kubernetes、Microservices。
- 团队方向：Catalogue、Customer、Pricing、Returns、Loyalty、Payments、Search & Recommendation、Partner / Marketplace、ZEOS（B2B fulfillment）、Fashion Store frontend。
- 期待：扎实 OOP / SOLID / 设计模式、对 microservices + REST / event-driven 有经验、熟悉至少一个 JVM 语言。

### 2.2 Machine Learning / Applied Science Intern
- 技术栈：Python、PyTorch / TensorFlow、Spark、Databricks、AWS SageMaker / EMR、Airflow、Kubeflow、自研 ML Platform；推荐与 CV 团队还使用 GPU 大量 embedding 训练。
- 团队方向：Recommendation、Search Ranking、Sizing & Fit、Fashion Image Understanding、Fraud、Marketing Mix Modeling、Pricing、Generative Fashion Assistant。
- 期待：MLOps 思维（CI/CD、监控、可复现 pipeline）、能从论文 → prototype → 上线全链路有理解。

### 2.3 Data Engineering / Data Science Intern
- 技术栈：Spark / Scala / Python、SQL（重度）、dbt、Airflow、Databricks、AWS（S3 + EMR + Glue + Athena）、Tableau / Looker。
- 团队方向：Data Platform、Business Analytics、Marketing / Growth Analytics、Customer / CRM、Logistics。
- 期待：强 SQL、熟悉 modern data stack、统计基础好、能讲清"问题 → 数据 → 实验 → 结论"。

## 3. 面试流程

### 3.1 通用结构（实习/初级）

1. **Recruiter Screen**（30 min）— 背景、动机、签证（关键："你是否已有 EU 工作权？"）、薪资期望。
2. **Codility Online Test** — 60–90 分钟，2–3 题，包含 1 道 Java/Python 算法题 + 1 道 SQL + 1 道 DSA；非常依赖 IDE 习惯。
3. **Take-home Assignment**（部分岗位有）— 设计 + 实现一个小服务（Java/Kotlin 微服务）或 ML 任务；时间通常 1 周内。
4. **Technical Interview**（1–2 轮，60 min）
   - SWE：项目深挖 + 中等 LC 题 + System Design（microservices / REST API / DB choice / 缓存 / Kafka）。
   - ML / DS：take-home review + ML 概念深挖（MLOps、A/B 实验、模型选择）+ 1 道编程 / SQL。
5. **Hiring Manager / Team Match**（45–60 min）— 偏行为 + 团队对齐。
6. **Final / Panel**（部分岗位）— 与跨团队 senior 一起聊文化 + 系统设计。

整体周期：**平均 32 天**（Glassdoor）。

## 5. 签证与跨境工作

- **EU/EEA 学生 / 居民**：自由工作；优先候选。
- **非欧盟在德国在读**：可凭 *Aufenthaltserlaubnis* 学生居留进行 **强制实习 (Pflichtpraktikum)** — 不计入 120 全日 / 240 半日 / 年 的工作时长上限；**自愿实习 (freiwilliges Praktikum)** 会计入。
- **海外申请者**：Zalando 明确**不为实习 / 学徒 sponsor 签证**；非欧盟候选人需先解决在德合法身份。
- **全职毕业生**：可走 EU Blue Card（2025 阈值约 € 45,300/年，紧缺职业稍低）；Zalando 经常 sponsor 正式岗位。
- **Helsinki**：与德国类似，需要 EU 工作权或 Finnish student residence permit；Future Talent Program 同样要求合法身份。

## 6. 申请策略 & 时间线

| 阶段 | 时间 | 备注 |
| --- | --- | --- |
| Job posting 上线 | 全年滚动，主要批次 11–3 月 | 通过 jobs.zalando.com 投递，**不接受邮件投递** |
| Application screening | 1–2 周 | Resume 关键词：Java / Kotlin / Microservices / AWS / Kafka / Spark / SQL |
| Recruiter call | 1–2 周 | 必问"是否已有 EU 工作权" |
| Codility | 1 周内完成 | 题目偏 OA 难度，注意时间 |
| Take-home（如有） | 1 周 | README + 单测 + CI 配置 = 评分加分 |
| Tech + HM | 2–4 周 | 跨多人面试，时间易延期 |
| Offer | 平均 32 天 (实习 18 天) | 1–2 周决定窗口 |

申请技巧：
- Zalando 是**德国 #1 工程 hub of Berlin tech scene**，每年实习申请人数极多，海外候选人显著吃亏（签证原因），请优先准备：
  1. 简历突出 Java/Kotlin + microservices + AWS / Kafka 项目；ML/DS 突出端到端项目 + MLOps。
  2. Codility 题型熟悉（在线练习 codility lessons 1–10）。
  3. Take-home 高质量：单测 + dockerfile + README 中"为什么这样设计 + tradeoff"。
- Helsinki Future Talent 是非常划算的入口：流程短、转正率高、海外申请友好度略高（仍需 EU/Finnish 签证）。
- Master Thesis 是另一个高效通道，特别针对 ML / DS。
- Werkstudent 适合已经在德国留学的学生，长期累积，毕业前可获 referral 转正。

## 8. 关键风险 / 提示

1. **不 sponsor 签证**：是 Zalando 实习最大的卡点；海外候选人需提前解决学生签 / Blue Card 家属签 / 自由工作权。
2. **流程偏长 + Codility 是硬门槛**：题量大、时间紧；不在 Codility 上熟练的候选人易失误。
3. **Take-home 评分严苛**：写得能跑还不够，需要展示工程素养：单测、README、设计 trade-off 说明、CI。
4. **组织庞大、流程慢**：32 天平均；很多人反馈 recruiter 沟通节奏拖沓 / 跨 squad 调度难。
5. **实习薪不高**：€1,500–2,500/月，Berlin 房租约 €600–1,200 单间，扣完税到手有限，需自行预算。
6. **AI / Fashion AI 方向加分明显**：2024–2025 在推荐 / sizing / generative assistant 上投入显著，简历 + 面试故事建议沾边。
7. **Praktikum vs Werkstudent 区分**：海外候选人需弄清自己是"强制实习 (Pflicht)" 还是"自愿实习 (Freiwillig)"，前者不计入 120 工作日上限。
