# DeepMind / Google DeepMind 实习情报 (London)

> 更新时间: 2026-05-26
> 信息丰富度: ⭐⭐⭐⭐ (流程/薪资数据充足, intern 转正具体数字不公开)
> 一句话定位: 全球顶级 AI Lab, Google Research 合并后的 "Google DeepMind", 实习是进入 LLM/AlphaFold/Gemini 圈层的最稀缺入场券

## 1. 基本信息

- **欧洲办公室 (本任务范围)**:
  - **London (主总部, King's Cross)**: 几乎所有 RS/RE intern 都在这里
  - 注意: Paris 也有小 DeepMind 团队 (Gemini multilingual / 部分 robotics), 但 intern 名额极少
  - 全球还有 Mountain View / Zürich, 但 EU 范围内 London 是主力
- **实习岗位类型**:
  - **Research Scientist Intern (RS Intern)**:
    - 重论文产出, 与 host researcher 一起做 publishable 工作
    - 强偏 PhD (CS / Math / Physics / Neuroscience / EE), Master/Bachelor 出现案例**极少**
  - **Research Engineer Intern (RE Intern)**:
    - 偏工程, 与 RS 协作搭实验框架、scaling experiments、infra
    - PhD 不强制, 强 MSc + 顶级 SWE 背景能进
  - **Student Researcher Program** (独立项目):
    - **12-24 周**, 最少 4 天/周, 在 Google office
    - 任意学历 (Bachelor/Master/PhD), 是 DeepMind/Google Research "灰度版"实习
  - **Research Ready Programme** (针对**英国本土本科生**的限定项目):
    - 8 周暑期, 由 Cambridge / York / Imperial / AIchemy 等大学联合 DeepMind 出资
    - **不是 DeepMind 直招**, 是大学项目, 但带 DeepMind brand
- **实习时长**:
  - RS/RE intern: **3-6 个月** (大多 6 个月, 与 PhD 学期空档同步)
  - Student Researcher: **12-24 周**
  - Research Ready: 6-8 周 (固定夏季)
- **招聘周期**:
  - **9-11 月开放** Summer 2025/2026 RS intern
  - **12 月-1 月** deadline (有些 team 提前关), 强 rolling 倾向, **早投早面**
  - Student Researcher: 全年 rolling, 但 Q4 最活跃
  - Research Ready (UK 校内): 1-2 月校内 deadline, 2-4 月通知

- **是否有独立 Graduate Programme**: **部分有**
  - **AI Residency / DeepMind Fellowship** (历史名称变动): **12 个月** cohort 项目, 类似 Google AI Residency 的 EU 版, 面向**应届 PhD / 强 Master with publications**;以"做出 1 篇顶会 paper"为产出目标, 完成后 fast-track 走 RS / RE full-time 内部转岗;London 主要, 名额极少 (年 < 20 人 EU 总量)
  - **RS / RE 直接 NG offer**: 大多数情况, 应届 PhD 直接面 RS full-time, 没有 cohort, 入职即 IC
  - **Student Researcher → NG conversion**: Student Researcher 在 DeepMind 完成 6-12 个月后, 强 perf 案例可直接转 NG offer (host endorsement 是关键)
- **非EU签证 (全职)**:
  - UK: **Skilled Worker visa** — DeepMind (Google UK Limited 旗下) 是 top UK Skilled Worker sponsor; RS / RE base £80K-£140K 远超 £38,700 threshold;**Global Talent visa** 也是 PhD 候选人通道 (走 Royal Society / Royal Academy of Engineering endorsement, 不依赖雇主, 灵活度更高 — DeepMind 有员工帮你 endorse)

- **面试流程差异 vs 实习**:
  - 轮数: 实习 5-6 阶段 → NG **6-8 阶段** (加 1-2 轮 RS / RE pair 的 second-opinion interview + 1 轮 hiring committee chat)
  - Research Interview: NG 比 intern 更深 — 要求候选人讲 **3-5 个未来 1-2 年要做的 research 方向**, 不再只是 PhD work deep dive (intern 只问 PhD)

  - Coding: RE NG 仍 LeetCode Medium + PyTorch tensor 题, 但加 1 道 production code review
  - Culture: DeepMind 强调 **alignment / safety** value, NG 期望对 AI safety 有 informed opinion (intern 此项 light touch)
  - Hiring committee: Google-style 委员会决策, 比 intern 流程更慢 (NG 通常 8-12 周才出 offer)
- **录取竞争**: **极卷**, **RE NG acceptance < 1%** (Sundeep Teki 数据, 与 intern 同级);**RS NG 几乎硬要求顶会 (NeurIPS/ICML/ICLR) 一作 + h-index 10+**。Host match 仍是事实门槛 — 没有 PI 愿意带的候选人即使技术过也挂。**国际生 friendliness**: ⭐⭐⭐⭐ (UK Skilled Worker + Global Talent 双通道, sponsor 标准化;海外 PhD 通过 visa 进入是标配)

## 2. 签证与国际学生政策

- **DeepMind 是 UK Skilled Worker 注册 sponsor**, 全职转正路径明确
- **实习路径**:
  - **优先**: 在英国就读的本科/硕士/PhD (Student visa 自动允许实习时长内 full-time)
  - **Graduate visa** 持有人 (毕业后 2 年开放期): 可直接做实习, 不需要新 sponsor
  - **GAE (Government Authorised Exchange)**: DeepMind 历史用过, 实习短期 (≤ 24 个月)
  - **Tier 5 GAE** 已并入 GAE 新规, 但本质相同
- **远程实习**: COVID 后部分 RS intern 允许 hybrid; 但 visa 角度仍需在 UK 境内
- **海外 PhD 临时 visit 做 intern**: 现在标准是发 GAE certificate of sponsorship, DeepMind 帮你提交
- **已知 sponsorship 案例**:
  - 大量 PhD intern (来自欧洲/北美/亚洲学校) 通过 GAE 走完 6 个月 — 公开 Quora / LinkedIn 上有案例
  - 全职转正几乎都用 Skilled Worker
- **国际学生友好度**: **⭐⭐⭐⭐** (在 AI lab 中算友好, 但**只对真正能进面试的人友好** — 简历筛非常严)

## 3. 面试流程

### 3.1 Research Scientist Intern (RS Intern) — 最难

**总轮数: 5-6 大阶段, 周期 4-6 周**

1. **HR / Recruiter screen** (30-45 min): 背景, research interests, **host match preference**
2. **Hiring manager / Host chat** (30-45 min): 论文讨论, 兴趣方向匹配
3. **Coding Round** (60 min × 1-2): LeetCode **Medium** 主导, 偶有 Hard; **PyTorch tensor shape 题** 是签名题
4. **ML / Math Round** (60-90 min): 
   - 数学基础 (linear algebra, eigenvalue/eigenvector, calc, prob)
   - ML 理论 (loss functions, regularization, optimization, bias-variance)
   - LLM 前沿 (Transformer 从零、attention 实现、finetuning 方法、Gemini/DeepSeek 训练栈细节)
5. **Research Interview** (60-90 min × 1-3 个 scientist):
   - **PhD work deep dive** — methodology, limitations, follow-up
   - 假设性 research question — "若给你 100 GPU, 你会怎么 attack X 问题?"
6. **Culture / Team Fit** (45 min)

题型分布 (RS Intern):
- Research / paper deep dive: **45%**
- ML 基础数学/理论: **25%**
- Coding (LeetCode + PyTorch impl): **20%**
- BQ / culture: **10%**

难度评分: **3.2 / 5** (Glassdoor RS), 62% positive, 平均 **45 天**

### 3.2 Research Engineer Intern (RE Intern) — 工程偏向, 更可达

**总轮数: 4-5 阶段, 周期 4-6 周**

1. **Recruiter call**
2. **Coding interviews × 2**: LeetCode Medium 为主, 偶有 system 题
3. **ML Fundamentals interview**: optimization, regularization, loss, transformers, **训练 + 推理实战题** (e.g., "how do you debug an exploding gradient?")
4. **ML Design / System Design**: large-scale training pipeline, distributed training (DDP, FSDP), checkpoint, sharding
5. **Lead Researcher interview**: team match + 项目讨论

题型分布 (RE Intern):
- Coding: **30%**
- ML 基础: **25%**
- ML / System design: **25%**
- 项目: **15%**
- BQ: **5%**

**Sundeep Teki guide** 报告: "DeepMind 工程岗 acceptance < 1%, 像 PhD defense + 工程 exam 的混合"

### 3.3 Student Researcher Program

- 流程较轻: 1-2 轮 + 与 host researcher 直接对话; 强依赖 host endorsement, 内推机制是核心
- 通常 host 提前认识候选人 (paper / workshop / Twitter)

### 3.4 准备路径 (David Stutz / 公开博客建议)

- 2-3 周冲刺: Cracking the Coding Interview + ML 论文复习 + tensor 操作
- 重温 textbook (Bishop PRML, MacKay, Goodfellow Deep Learning Book)
- 重点: **多头注意力从零实现** + **训练 debug 故事**

## 6. 申请渠道与建议

- **官网投递**:
  - https://deepmind.google/careers/ (主入口)
  - https://deepmind.google/student-researcher-program/ (Student Researcher 单独通道)
  - 历史 Greenhouse 接口: https://boards.greenhouse.io/deepmind/ (后转向 Google careers)

- **内推途径 (DeepMind intern 最强力路径)**:
  - **Twitter / X**: DeepMind researcher 普遍活跃, 给他们看你 paper/notebook 是有效 outreach
  - **NeurIPS / ICML / ICLR workshop**: DeepMind paper 一作的 spotlight, 现场认识 host
  - **学术导师推荐**: PhD 导师与 DeepMind 团队有合作的, referral 非常重要
- **简历/背景要求**:
  - **RS Intern**: PhD 在读 + **顶会一作论文** (NeurIPS / ICML / ICLR / Nature 子刊) 是事实门槛; 偶尔接受顶尖 Master 但凤毛麟角
  - **RE Intern**: Master + 工程实力 (Kaggle top / GitHub 强项目 / 之前 FAANG SWE intern) 可以打; PhD 也很常见
  - **Student Researcher**: 学历宽松, 但 host endorsement 决定一切
  - **强 baseline**: 至少 3 篇 arXiv preprint + 顶会 1-2 篇, 或 1 篇 spotlight/oral
- **强烈推荐**:
  - 在 deepmind.google 上看每个 hiring manager 的 publication list, 申请前**针对性地**讲为何契合 ta 的研究
  - DeepMind 招聘 portal 允许 "preferred host", 务必填写

## 7. 踩坑与社区评价

- **publication 几乎是 RS intern 的硬要求**: 没有顶会一作很难过 HR 筛 (网传 Hacker News 也明确大厂 PhD intern 多有 NeurIPS/ICML 一作)
- **acceptance < 1%** (RE intern, Sundeep Teki 数据), 比 Apple / Nvidia intern 都更严苛
- **流程长 + feedback 少**: Glassdoor 多次提到 "lack of feedback" 是常态; 挂在哪轮经常不告知
- **Host match 风险**: 即使技术面全过, 没有 host researcher 愿意带你也会被挂; **多 host preference + 提前沟通**关键
- **转正率 (Intern → Full-time)**:
  - 公开数据**没有**, 但社区估算: **PhD RS intern → 留下做 RS** 比例**远低于** 20-30% (大多还得 PhD 毕业 + 重走 full-time 流程)
  - **RE intern → RE full-time** 转正率高于 RS, 但仍非自动 — 需要 hiring committee 单独评审
  - Quora 上有"DeepMind 在 CV 上等同于六位数 offer 直接拿"的说法是**夸张** (但 brand 效应确实强)
- **PhD intern 周期长**: 6 个月几乎是 PhD program 一个学期的牺牲, 需要导师同意
- **Disaster 案例**: 公开博客 "My DeepMind Interview was a Disaster" 描述了即使有论文背景也被挂 — interviewer 严苛度极高
- **国际学生友好度**: ⭐⭐⭐⭐ (海外 PhD 通过 GAE 进入是标准路径)
- **DEI 倾斜**: DeepMind 在 underrepresented group 推广上有专门项目 (Research Ready 即一例), 但**不放松技术门槛**
