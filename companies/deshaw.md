## 2. 签证与国际学生政策

- **UK Skilled Worker 2026 阈值**: base £41,700 / going rate. D.E. Shaw London 任何 SWE / Quant 岗都远超.
- **CoS**: A-rated sponsor licence, 历史发 CoS 稳定 (但总人数少 = 配额其实不是问题, bar 才是).
- **实习 visa 路径**: 英校 Student visa OK; 非英校 PhD 候选人公司可帮 sponsor Tier 5 Government Authorized Exchange 或 Standard Visitor PPE.
- **Graduate Route (PSW)**: 英校毕业 2 年 PSW, 之后转 Skilled Worker.
- **特殊**: D.E. Shaw "Research" (生物计算 + drug discovery, NYC) 与对冲基金主体 (D.E. Shaw & Co.) 是分开的实体, 申请前确认 entity. London 全是 HF 主体.
- **Reference check 严格**: visa 申请通过率高, 但前面的 ref check 极挑, 离职推荐人质量很关键.

## 3. 面试流程

| 轮次 | 内容 | 时长 |
|---|---|---|
| 1. CV / Application Review | Recruiter screen + 学校 + 量化背景 | 1-2 周 |
| 2. OA (Online Assessment) | HackerRank / Codility, 4 道 LeetCode medium-hard + 概率/统计概念题 | 90-120 min |
| 3. Phone Interview | 1 名 engineer, 数据结构 / 算法 / 项目 deep-dive | 60 min |
| 4. Virtual Interviews | 2-4 轮, 含 coding (Python/C++/Java) + 数学/概率深度 + system design + behavioral | 半天-一天 |
| 5. Case Study / Code Sample | 部分岗位要求 writing sample 或 code sample 提交 | varies |
| 6. Reference Calls | D.E. Shaw 主动联系你提供的 references | 1-2 周 |
| 7. Offer | Recruiter 沟通 final | - |

平均总时长 ~14 天 (Indeed/Glassdoor 数据) 但实际拖 4-8 周常见. OA 报告: "4 LeetCode medium-hards" [来源 (搜索结果): https://www.geeksforgeeks.org/gfg-academy/de-shaw-recruitment-process/].

## 6. 申请渠道与建议

- **官网投递**: [deshaw.com/careers](https://www.deshaw.com/careers) — London 岗位明确标 "(London)"; intern 走 [campus.deshaw.com](https://campus.deshaw.com/internships).
- **校招 deadline 警惕**: 官方页明确写 "selection is completed early in Q1 of the internship year, recommend applying by end of prior calendar year". Summer 2027 申请窗口 Sept-Dec 2026.
- **内推**: D.E. Shaw 极依赖网络 + 学校 target list (Oxford/Cambridge/Imperial/UCL 英校, MIT/Princeton/CMU US 校). Cold apply 通过率最低, 学校 careers fair / target school events 最高.
- **CV 重点**:
  - 顶校 + 高 GPA (硬性, 几乎没例外)
  - 强数学 / 概率背景 (Putnam, IMO, Kaggle Top)
  - C++ / Python performance optimization 经验加分
  - PhD 候选人优先 Quant Analyst / Systematic Research 路径
- **同档备选**: Citadel, Citadel Securities, Jane Street, HRT, Jump, Two Sigma — 这些是顶 tier 同时面 set. Squarepoint / Marshall Wace 在 London 也常 cross-apply.

## 7. 踩坑与社区评价

- **Bar 极高**: London 招聘人数少 + 数学/概率深度极强, 顶 EU 校 + Putnam / IMO 级别经历仍可能挂.
- **流程慢且不透明**: 官方说 14 天平均, 实际多人报 4-8 周 + 多次 ref check.
- **Code sample / writing sample 要求**: 部分 senior 岗会要求, 准备一份 polished GitHub repo.
- **没有正式 grad cohort**: 不像 G-Research / Man Group 有结构化 Programme, 直接进入 team, 学习曲线陡.
- **AI tool 禁用** (类似 Two Sigma 政策, 面试是真人监督).
- **Glassdoor 评价**: 4.1/5, 高度评价 colleagues quality + comp, 抱怨 work intensity + 不透明的 bonus 决策.
- **国际学生友好度**: ⭐⭐⭐ — sponsor 稳定但 bar 极高, 招聘量小是关键瓶颈. PhD 候选人比 undergrad 通过率显著高.
- **不要混淆**: D.E. Shaw Research (drug discovery, NYC only) ≠ D.E. Shaw & Co. (HF, London+NYC). 申请前确认是 HF entity.
