# Millennium (London) — 实习 + Grad 情报

> 全球最大的 multi-manager / pod-based HF, AUM ~$73bn. London 是 Millennium International 主体 (欧亚分部, 800+ 员工). pod-based 意味着每个 PM 自带团队, 招聘高度去中心化 — 各 pod 招聘节奏 / 流程 / 薪酬独立, **没有统一 Grad Programme**.

---

## New Grad / Junior 岗位

> 信息丰富度: ⭐⭐⭐ (pod-by-pod 不透明)

- **常见岗位名 (London-active)**:
  - **Quantitative Developer** (pod-embedded, 多个 pod 独立招)
  - **Senior Platform Engineer** (central IT Infrastructure)
  - **Corporate Technology Support Engineer** (Equity IT)
  - **Quant Modeling Intern** (Summer cohort, 有限)
  - **Summer Quantitative Finance Internship** (~24-hr Jupyter take-home)
  - Cubist (Point72 quant sub-unit) 是 **sibling 对比** — 类似 pod-based 但 Cubist 是 Point72 内部 systematic 部门, 不是独立 firm. Cubist KEPL elite team intern $25k/月 (NYC, USD).
- **欧洲地点**: London (主); 部分 corp / IT 在其他 EU
- **是否有独立 Graduate Programme**: **没有统一 cohort programme** — 每个 pod 或 central tech team 单独招. 与 G-Research / Man Group 形成鲜明对比.
- **非EU签证 (全职)**: UK Skilled Worker £41,700 / SOC 2134 ~£49,400. Quant Dev base 通常 £100k+ 远超阈值. Sponsor licence A 级.
- **薪资结构 (pod-dependent, 高方差)**:
  - **Intern**: Summer Quant Modeling intern London estimate £6-10k/月 (官方未公开, 1point3acres 报告 24-hr Jupyter project + 普通 corporate housing)
  - **全职 Quant Developer (London)**:
    - Junior: base £100-140k + bonus 30-100% (pod-dependent)
    - 中级到 senior: total £200-400k+ 常见
    - 个别 senior Quant Dev 报告 ~£300k+ all-in 稳定数年 [来源 (搜索结果): https://www.wallstreetoasis.com/forum/hedge-fund/quant-developer-in-a-pod-career-advice]
    - Levels.fyi Lead SWE $500k 顶档
  - **Quant Researcher / PM**: $2m+ 不少, 但绝大多数 < $300-400k all-in [来源 (搜索结果): https://youngandcalculated.substack.com/p/the-detailed-and-honest-reality-of]
- **申请窗口**: pod-by-pod rolling; Summer intern 通常 9-11 月开放; central Platform / IT 全年滚动招.
- **面试流程差异 vs 实习**: pod 面试 ad-hoc 由 PM 决定, intern OA 标准化 (90 秒/题 sequences / combinatorics); FT 更深入 (stochastic calculus, C++).
- **录取竞争**: 入职容易 < 顶级 firm (HRT/Jane Street), 但 pod 1-2 年内 P&L 不达标可能 被解散.

## 2. 签证与国际学生政策

- **UK Skilled Worker 2026 阈值**: £41,700 / SOC code going rate. Quant Dev / Platform Engineer 起薪远超.
- **CoS**: A-rated sponsor licence, pod hire 通常 sponsor 无问题 — 因为是 pod 自己付的 cost.
- **实习 visa 路径**: 英校 Student visa; 国际学生 Standard Visitor PPE 或 Tier 5.
- **Graduate Route (PSW)**: 英校毕业 2 年 PSW 适用.
- **pod-dependent**: 不同 pod 对国际学生态度不同 — 部分 pod 偏好已有 right-to-work 的候选人 (避免 visa hassle), 部分 pod 完全无差别. 申请前问清楚 pod 的 sponsorship history.
- **Cubist (Point72) 对比**: Cubist 在 London 也有 office, 走 Point72 careers portal, sponsor 政策类似.

## 3. 面试流程

pod 流程差异巨大, 以下是 community 报告的常见模式:

| 轮次 | 内容 | 时长 |
|---|---|---|
| 1. OA (intern path) | Sequences test + Zap-N + 组合/概率, 90 秒/题极速 | 60-90 min |
| 2. Take-home Project | 24-hr Jupyter notebook: price making / beta-adjusted hedging / 简单 arbitrage 用 pandas | 24 hr |
| 3. Technical Interview | 概率 / stochastic calculus / C++ 选项定价 + LeetCode-style | 60-90 min |
| 4. LeetCode Round | 1-2 道 medium-hard | 45-60 min |
| 5. Talk with Seniors | PM / lead 聊 fit + 项目 | 45-60 min |

平均 41 天 hire-to-offer (Indeed) — pod 决策快但 reschedule 频繁. Pod-specific Q 完全不可预测 (每个 PM 自定义).

## 6. 申请渠道与建议

- **官网投递**: [career.mlp.com](https://career.mlp.com/careers) — pod 招聘和 central Tech 都在这里, 搜 "London" filter.
- **内推**: 极关键 — pod 招聘高度依赖 PM 个人网络, 找到目标 pod 的 Quant Dev 内推通过率显著高于 cold apply.
- **CV 重点**:
  - C++ 高性能 + Python 数据处理双语都强
  - kdb+/q 是 huge plus (很多 equity pod 用)
  - 量化项目 (Kaggle, 自营策略 GitHub) + 数学背景
  - **避免 generic SWE CV** — 突出 quant-adjacent (期权定价, time series, ML alpha)
- **Cubist (Point72 quant unit) sibling 对比**:
  - Cubist 也是 pod-based, 但是 Point72 内部 systematic 部门 (不是独立 entity)
  - Cubist KEPL (elite Python quant team) intern 给 $25k/月 (NYC)
  - Point72 整体 intern $130-165k annualized (非 KEPL)
  - 流程 Point72 比 Millennium 标准化 (Cubist Quant Academy 有 1 月 cohort)
- **同档备选**: Citadel, Balyasny, ExodusPoint, Point72 (Cubist), Squarepoint, Marshall Wace, Brevan Howard — multi-strat pod-based 同档.

## 7. 踩坑与社区评价

- **流程不透明**: 各 pod 招聘节奏 / 流程 / 薪酬独立, 准备工作很难 generalize. 不要找以前学长经验 "复用".
- **Pod risk**: 入职后如果 pod P&L 不达标 (通常 -5% drawdown), 整组解散, 你也 OOD. 需要心理准备.
- **薪酬 stagnation**: WSO 有报告 pod 间换 base 不涨 (~$400k 几年不动), 增长靠 bonus + 进顶 pod.
- **OA 严防作弊**: 90 秒 / 题 + ProctorU 监督 + sequences test, AI 工具几乎不可能用.
- **没有 cohort / mentorship**: 直接进 pod 单干, 缺 grad programme 的 onboarding 体系.
- **Glassdoor 评价**: 4.1/5, 高薪 + intense + politics 多 (pod 间资源竞争).
- **国际学生友好度**: ⭐⭐⭐ — sponsor 没问题但 pod-dependent, 部分 pod 倾向 right-to-work 已有的候选人. PhD / 顶校量化背景显著加分.
- **Cubist 对比 takeaway**: 如果你想要 pod-based 但希望多一些结构, Cubist Quant Academy (Jan 2026 cohort) 比 Millennium 体系化.
