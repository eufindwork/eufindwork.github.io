## New Grad / Junior 岗位

> 信息丰富度: ⭐⭐⭐⭐⭐ (Levels.fyi London L1 直读 + eFinancialCareers headcount + Wikipedia 全球)

- **常见岗位名 (区分关键)**:
  - **Software Engineer** (London): 通用 OCaml SWE, 包含 platform / trading systems / research infra — 不一定碰 strategy; 这是 SWE 路径主入口
  - **Quantitative Trader / Trading**: 数学 + 心算主导, 非 SWE 路径
  - **Quantitative Researcher**: ML/统计/PhD heavy
  - **Strategy and Product**: 偏 business
  - **Machine Learning**: ML 工程 / 应用
  - 注: Jane Street 没有 "Quant Developer" 这个 title — SWE 直接对接 trader, 工程 + 业务深度高 [来源 (实际打开过): https://www.janestreet.com/join-jane-street/internships/]
- **欧洲地点**: London (**唯一欧洲办公室**); 其他全球 NY / Hong Kong / Singapore
- **是否有独立 Graduate Programme**: **没有"品牌化 grad scheme"** (区别于 Citadel/HRT 的 named programme); 全职 New Grad 直接走 "Software Engineer, London" open role 投递 → 入职后等同其他 SWE; 实习转正是主要 pipeline (intern → return offer FT)
- **实习项目细分 (London 可投)**:
  - **Software Engineering Internship** (10-12 周, May-Sep, **London / NY / HK 都开**): 主流路径; 全公司 SWE intern Levels.fyi 显示时薪 **$144.23/hr ≈ $4,800/周** (NY 基准, London 按地理 15-25% 折扣估 £3,500-£4,500/周 + 公司报销 housing) [来源 (实际打开过): https://www.levels.fyi/internships/Jane-Street/Software-Engineer-Intern/]
  - **Quantitative Trading Internship** (London 可投)
  - **JSIP (Jane Street Immersion Program)**: **仅 NYC**, 不在 London! Mid-June - Mid-August, 一二年级本科 + 多元背景, $12,500 scholarship + 包 housing/travel/meals, 2026 cycle 截止 2026-02-08 — 这是品牌混淆点, JSIP ≠ 普通 SWE intern [来源 (实际打开过): https://www.janestreet.com/join-jane-street/programs-and-events/jsip/]
- **非EU签证 (全职)**: **Sponsor 积极** — Jane Street UK 在 worksponsors.co.uk 列入 active Skilled Worker license; L1 SWE 全职 base ≈ **£175-234K** (远超 2026 SOC 2136 going rate £55K 阈值 + Skilled Worker 全国阈值 £41,700, 完全无障碍); intern 是 Tier 5 GAE / Student visa work, FT 转 Skilled Worker 是标准操作
- **薪资结构 (London, gross/年)**:
  - **Intern (SWE)**: $144.23/hr (≈ £113/hr 当前汇率) 全球统一基准, London 实际可能略低; 11-12 周累计 ~$60-70k stipend + corporate housing + 报销机票; quant/trading intern 周薪 $4.5-6k+ [来源 (实际打开过): https://www.levels.fyi/internships/Jane-Street/Software-Engineer-Intern/]
  - **全职 New Grad SWE (London L1)**: **TC ~£245K = $327K** (Levels.fyi 直读); base $234K + bonus $91.1K + stock $1.8K (Jane Street 是私人公司, "stock" 是名义数字 / 实质为 phantom equity) [来源 (实际打开过): https://www.levels.fyi/en-gb/companies/jane-street/salaries/software-engineer/locations/london-metro-area]
  - **Quant Trader / Researcher New Grad**: TC 跨度 £300K-£500K+ (eFinancialCareers 报道 partner-level 达 £14m, "UK average pay £808k per head" — 这是 partner 拉高的均值)
  - **奖金主导**: base 占 TC ~70%, 剩余是 discretionary year-end bonus; 公司私人, 无 RSU 上市抛售路径
- **申请窗口**: SWE intern + FT rolling, 但 fall recruiting (Sep-Nov) 是最佳窗口; JSIP 2026 截 2026-02-08
- **面试流程差异 vs 实习**: 实习面试 ~3 轮 (HR + 1-2 tech), FT ~5-6 轮 (HR + 2-3 tech + final 多人 + culture/judgment); 都包含 **mental math 50q/8min** + probability puzzles + OCaml/任意语言 coding; **可以用任意语言面 — 官方明确"不建议第一次试 OCaml"** [来源 (搜索结果): https://www.janestreet.com/preparing-for-a-software-engineering-interview/]
- **录取竞争**: London ~30-60 SWE FT 每年 (按 41 technologist 净增量推断); intern → FT return offer 命中高 (Jane Street 严重依赖 intern pipeline)

## 2. 签证与国际学生政策

**核心结论: Jane Street London 在 Skilled Worker license 名单, 全职 sponsor 标准操作; intern 走 Tier 5 GAE 或 student work 路径**.

- **Skilled Worker (全职)**: Jane Street UK Partnership LLP / Jane Street Europe Limited 持 sponsor license (worksponsors.co.uk 确认); L1 base £175K+ 远超 2026 全国阈值 £41,700 与 SOC 2136 going rate £55K
- **实习签证**: UK student 直接用 Tier 4 visa 自带 work right (full-time vacation work); 非英国学生走 Tier 5 GAE 或 Standard Visitor (取决于学校所在国 — 美国学生常见)
- **relocation 包**: 提供机票 + 临时住房 + 报销搬家费 (intern 完全免费 corporate housing; FT 有 relocation package)
- **国籍限制**: 无明显限制, 但 trading/research 涉及 regulated function 时背景调查更严 (FCA SMCR)
- **EU 27 申请人 (Brexit 后)**: 与非 EU 同走 Skilled Worker; 无 settled status 也可
- **回家工作 / remote**: London office 强制 onsite (4-5d/week), 不接受跨国 remote
- **从 London 转 NY/HK**: 内部 transfer 走 L-1 (NY) 或 employment pass (HK), 常见 path 但需 2+ 年 tenure

## 3. 面试流程

| 轮次 | 内容 | 时长 |
|------|------|------|
| 0. 申请 | janestreet.com/join-jane-street/ 直投; 不强制内推 (Jane Street 不办 referral 文化) | - |
| 1. 在线测试 | **50 questions in 8 minutes** mental math 测试 (probability + 心算, 例: 2 个骰子点数和 ≥10 概率) — Jane Street 最有名的关 | 8 min |
| 2. 电面 (recruiter) | 经历 + 兴趣 + 项目 | 30 min |
| 3. 技术电面 1 | 算法 + 概率 + market making puzzle; **可用任意语言 (Python/C++/Java/OCaml 都行)**, 官方说"不要第一次试 OCaml" | 45 min |
| 4. 技术电面 2 | 更深算法 (DP / recursion / 数据结构) + probability puzzle 升级 | 45 min |
| 5. Final / onsite (London 或 NY) | 4-6 轮: coding (OCaml or own) + probability + game theory + culture/judgment + lunch with engineer; SWE 路径加 system design (low-latency / message bus) | 5-6 hr |
| 6. Offer | 通常 1-2 周, 整体 4-6 周 application → offer |

**Jane Street 特色**:
- **Mental math 50/8** 是入门关, 不通过直接拒
- 强调 **clever insight over rote computation** (面试官奖励"看出 trick"而非暴力解)
- 面 OCaml 可选, 但 functional programming 思维 (递归 / immutability / pattern matching) 加分
- **没有 LeetCode hard 套路**, 题目更像 Putnam / Project Euler

## 6. 申请渠道与建议

- **官网投递**: https://www.janestreet.com/join-jane-street/open-roles/ — Jane Street 几乎只看官网申请
- **内推**: **不存在**或不流行 — Jane Street 反 referral 文化, 申请人公平对待; 不要花精力找内推
- **CV 重点**:
  - **Codeforces / IOI / Putnam / IMO** 显著加分 (Jane Street recruiter 直接看竞赛 ID)
  - GPA + 学校 prestige 重要 (Oxbridge / Imperial / UCL / Cambridge / Edinburgh top targets, 海外 Harvard/MIT/CMU/Princeton)
  - 开源贡献 (尤其 OCaml ecosystem: dune, async, core) 是巨大加分
  - 不需要 finance 背景, 不要硬塞 finance project
- **同档备选**: HRT (algo dev / SWE) / Citadel Securities / XTX (Software Dev) / Optiver / IMC / Susquehanna (SIG)

## 7. 踩坑与社区评价

- **50/8 mental math 是硬门槛**: 没准备过的人 90% 挂在这关; 准备建议是刷 Heard on the Street + 用 Anki 训练心算
- **OCaml fear is overrated**: 官方反复强调"不要第一次试 OCaml", 但部分 1point3acres / Blind 用户认为"会 OCaml 显著拉好感"; 折中是了解 functional concept 但用 Python 面
- **流程慢但 thoughtful**: 4-6 周 application → offer, 比 Citadel 慢, 但拒信通常会给具体反馈
- **国际学生友好度 (UK 签证)**: ⭐⭐⭐⭐⭐ — Skilled Worker license + 主动 relocation, L1 base 远超阈值; **Brexit 后 EU 学生与非 EU 同待遇**
- **薪酬满意度极高, growth 中等**: 集体奖金 = 内部 PnL 透明度低, 个人不知道自己"值多少钱"; senior engineer 跳槽难 (没有上市股权"金手铐", 但市场上没几家能匹配 Jane Street 薪资)
- **OCaml lock-in**: 离开 Jane Street 后, OCaml 经验在外部市场 niche; 但反过来 Jane Street 离职率极低 (~3-5%/yr)
- **Glassdoor 4.3/5** (全公司), 但 London 数据点少, 主要美国员工反馈
